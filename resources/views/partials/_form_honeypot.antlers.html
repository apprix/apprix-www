{{# @name Form honeypot @desc Shared spam honeypot for every public form. Invisible to humans; bots fill it and Statamic silently rejects the submission (each form declares `honeypot: fax` in its YAML). Offscreen rather than display:none, which some bots skip. Pairs with the time-trap in app/Providers/AppServiceProvider.php. Params: - id: unique input id (forms can coexist on one page). - model: Alpine x-model expression. Set it ONLY on alpine:form forms so the value rides along in the precognition form object. FormData forms (native POST / fetch) submit the raw input and need no model. #}}
