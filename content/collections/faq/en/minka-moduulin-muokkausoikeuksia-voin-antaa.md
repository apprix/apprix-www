title: 'What module edit permissions can I grant to a facilitator in the Administrate tool?'
answer: '<p>Edit permissions for a facilitator are granted as follows:</p><p>First set the value "On" for the parameter "Module''''s accessible parameter".</p><p>Then set the value of the Default Module Access parameter:</p><ul><li>"Edit": if the facilitator can edit all modules.</li><li>"View": if the facilitator can preview and duplicate all modules but not edit them.</li><li>"View": you can define the facilitator''''s edit permissions for one or more modules by clicking the "Settings" button under "User''''s Module Edit & Stats Permissions". When the button is clicked, a window opens. The window lists all modules. You can grant the facilitator edit permissions (edit, view, none) for one or more modules. In this window you can also grant statistics viewing permissions.</li></ul>'
