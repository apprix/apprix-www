answer: '<p>The Guard by IP setting is enabled in the main module. If a user tries to view the module or otherwise access it from an IP address that is not allowed, access is denied.</p><p>Solution: If the course needs to be completed outside the restricted IP addresses, the Guard by IP setting must be turned off.</p>'
