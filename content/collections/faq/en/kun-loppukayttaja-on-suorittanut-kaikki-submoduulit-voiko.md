title: 'Once the end user has completed all submodules, can they still browse the content?'
answer: '<p>Yes, if it is necessary to browse the submodule content after completing it, the Start New / Browse feature can be used.</p><p>With this feature, the end user can:</p><ul><li>either complete the course again,</li><li>or browse the content of an already completed course.</li></ul><p>At the beginning of the course the user can choose the option to "Browse the course you have completed", which allows them to view the course content without generating new entries in the statistics. Additionally, by pressing "Browse – click to skip" the user can browse through elements and skip tasks.</p><p>Find more information here: 8.6 Start New/ Browse <a href="/en/help/86-start-new-browse">/help/86-start-new-browse</a></p>'
