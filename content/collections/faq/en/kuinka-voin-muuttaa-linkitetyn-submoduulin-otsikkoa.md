title: 'How can I change the title of a linked submodule?'
answer: '<p>Check whether the main module is published. If it is published, unpublish it.</p><p>In the main module''s Edit Module dialog, click the pink Browse Submodules link. A list of submodules will appear.</p><ul><li>Click on the submodule to be updated. If the submodule is published, unpublish it.</li><li>Click the green Published button (the submodule status will now become unpublished). After this you can unlink the submodule from the main module.</li><li>Click the black Submodule Settings button.</li><li>Click \"Change to normal module\".</li></ul><p>The submodule has now been converted to a normal module.</p><p>Remove the title from the Intro element (the module''s first element) of the module you just converted (recently changed from submodule to normal module) and then update the titles of other elements as needed.</p><p>When all module titles have been updated, click the module''s Submodule settings and then Change to Submodule.</p><p>The module has now been converted back to a submodule.</p><p>The Change to submodule dialog opens. Find the main module to which you want to link the submodule, and click its pink Submodule Menu button.</p><p>The Submenu settings dialog opens.</p><p>Link the submodule to the main module by clicking the grey \"Select\" button. The grey button turns green when you hover over it.</p><p>Note! If the Select button is pink, this means another submodule is already linked to that button.</p><p>You can also update the element titles afterwards if needed.</p><p>Republish the submodule by clicking on the submodule name and then the black Publish button.</p> <img src="/files/docs/2024/04/HeaderTitle1.png" alt="" width="582" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;"> <img src="/files/docs/2024/04/HeaderTitleRemoveTitle2.png" alt="" width="771" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;"> <img src="/files/docs/2024/04/HeaderTitle3.png" alt="" width="641" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;"> <img src="/files/docs/2024/04/headertile4.png" alt="" width="627" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;"> <img src="/files/docs/2024/04/HeaderTitle5.png" alt="" width="859" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;"> <img src="/files/docs/2024/04/HeaderTitle6.png" alt="" width="709" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;">'
