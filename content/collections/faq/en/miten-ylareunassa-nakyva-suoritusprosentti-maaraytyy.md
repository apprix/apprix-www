title: 'How is the completion percentage shown at the top determined?'
answer: '<p>An online training created with Builder has a main module (e.g. "Site Orientation"), which can contain several submodules (like chapters of a book), such as in this case "Submodule 1 – Security issues", "Submodule 2 - Contact persons" and so on.</p><p>Builder starts calculating the completion percentage of the training in the main module from 0%.</p><p>(Site Orientation module in Builder: main module + submodules)</p><p>When the course participant arrives at this page (Submenu buttons element) and clicks one of the blue buttons on the right side of the window, they open a submodule (e.g. "Submodule 1: Security issues").</p><p>(This is the Submenu buttons element in the main module)</p><p>Builder starts calculating the percentage again because this is a new submodule. As shown in the image, the submodule does not contain a Finish element.</p><p>The percentage in each submodule starts from 0% again. In submodules the last element is always "Back to menu", but it is never shown to the course participant, so "100%" does not appear in the submodule.</p><p>When the course participant has completed all required submodules, they return to the Submenu buttons element again. All required submodules are marked as completed and they can continue the main module. The percentage continues normally in the main module because this is a different module from the submodules.</p><p>In other words, the main module and submodules are calculated separately.</p><p>(The Finish element is the last element of the main module)</p> <img src="/files/docs/2022/12/Koulutusprosentti-1-kuva.png" alt="" width="513" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;"> <img src="/files/docs/2022/12/Koulutusprosentti-2-kuva.png" alt="" width="673" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;"> <img src="/files/docs/2022/12/Koulutusprosentti-3-kuva.png" alt="" width="545" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;">'
