title: 'How can I add files and links to a module?'
answer: '<p>By following these instructions you can add links and files to the module that are available to the participant throughout the entire module completion:</p><p>Hamburger menu</p><p>The hamburger menu button is located in the top right corner of elements. Clicking the menu opens a list of files and links available to course participants.</p><p>Editing instructions</p><p>When you click the hamburger menu, the Link menu settings pop-up window opens. In this window you can add links (https://www.google.com/) and files (pdf, doc/docx, xls/xlsx, ppt/pptx, txt/csv, zip) to the hamburger menu.</p><p>Note!</p><p>Links and files that</p><ul><li>are added or removed through the hamburger menu of any element update automatically in every element of the module.</li><li>are added to the main module do not automatically update in submodules or from a submodule to the main module and other submodules, as the hamburger menu is module-specific.</li></ul><p>+ ADD LINK button</p><p>Click the + ADD LINK button when you want to add links to the hamburger menu. The Browse Media Library pop-up window opens. In this window you can see a list of previously added links, if any. If the link you are looking for is not yet in the window list, click the orange Add new link… button. A new pop-up window opens asking you to provide a name for the link. This name will appear as the link name in the window list. Enter a name for the link (e.g. google.com) and click OK. Another new pop-up window opens asking you to enter the link address, e.g. https://www.google.com, and press OK. The window closes. Select the ADD LINK button again. Select the link and the Insert selected button from the bottom right corner of the window. The link is now available from the hamburger menu.</p><p>+ ADD FILE button</p><p>Click the + ADD FILE button when you need to add a file to the hamburger menu. The Browse Media Library pop-up window opens. In this window you can see a list of previously added files, if any. If the file you are looking for is not yet in the window list, click the orange Add new file… button.</p><p>Note! Check the Show uploaded file in library box if you want the file to appear in the window''s file listing, i.e. the file library.</p><p>The File Explorer window opens. Select the file you want to add to the library. A new pop-up window opens asking you to provide a name for the file by which it can be found in the library. Enter a name for the file and press OK. The pop-up window and Browse Media Library window close.</p><p>Click the + ADD FILE button again in the Link Menu Settings window. Select the file in the Browse Media Library window and click the Insert Selected button at the bottom right of the window. The file now appears in the Link Menu Settings window. Select Done if you do not want to add any more files. The file has now been added and is visible in the hamburger menu.</p><p>The files and links added to the Link Menu Settings menu are now viewable from all module elements via the hamburger menu. If you want to remove a file or link, click the bin icon to the right of the link/file name.</p><p>If you want to remove the link and/or file from the list visible in the Media Library window, click on the file/link and the Hide selected file/link from library button at the bottom of the window. Note that the link/file will then only be removed from the list visible in the Media Library window, but not from the module, if the link or file has been added from the module element''s Link Menu Settings pop-up window.</p> <img src="/files/docs/2020/11/hamburger1-1000x353.png" alt="" width="489" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;"> <img src="/files/docs/2020/11/hamburger2.png" alt="" width="454" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;"> <img src="/files/docs/2020/11/hamburger3.png" alt="" width="394" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;"> <img src="/files/docs/2020/11/hamburger4-1000x595.png" alt="" width="442" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;"> <img src="/files/docs/2020/11/hamburger5.png" alt="" width="411" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;"> <img src="/files/docs/2020/11/hamburger6-1000x589.png" alt="" width="416" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;"> <img src="/files/docs/2020/11/hamburger7.png" alt="" width="357" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;"> <img src="/files/docs/2020/11/hamburger8.png" alt="" width="343" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;">'
