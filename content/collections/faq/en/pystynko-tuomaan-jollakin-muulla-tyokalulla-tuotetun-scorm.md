title: 'Can I import a SCORM course produced with another tool into the Builder environment and publish it there?'
answer: '<p>It cannot be done entirely independently, but it can be done with our assistance. We create a SCORM element for you and place inside it the course content produced with another tool. In practice, the Builder online course will then contain:</p><ul><li>an Intro element at the beginning (and possible user authentication),</li><li>next the aforementioned SCORM element, which contains the course content encapsulated in its entirety as a single package,</li><li>and at the end a Finish element, which transmits completion data to Builder statistics.</li></ul><p>Often the courses work directly after installation, but in some cases they may have links to somewhere else that need to be fixed before deployment.</p>'
