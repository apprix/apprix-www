title: 'What effects does it have if I change the login method for a training?'
answer: "<p>After the change, the user will log in with the new login method and may need to enter different information than before.</p><p>When changing the login method, the criteria used to search for previous completions also change.</p><p>This may mean, for example, that a completion started with the previous login method may not be found when the user logs in with the new login method, because the logged-in user''s details do not match the details of the previously started completion.</p><p>In a Landing Page training, completions are searched using the criteria specific to the Landing Page training''s login method.</p><p>If no completions are found for the linked training using the criteria specific to the Landing Page login method, the Landing Page will indicate that the linked training has not been completed.</p>"
