title: 'What logic determines where in the module the participant is directed when they open the training again?'
answer: '<p>The module works so that when logging in with certain credentials, the training starts and continues until the user reaches the last page (Finish element).</p><p>If the user stops the module completion midway, the next time they return the completion will continue from the same page where they left off.</p><p>If the module has submodules and a submodule has not been completed, the user must start the submodule from the beginning.</p><p>If the module has a Pass/Fail element and the user does not pass the final test, and the number of attempts has not exceeded the allowed maximum, they are automatically directed to the beginning of the final test (the element specified in Pass/Fail settings as "Retry from here", i.e. the beginning of the final test). Otherwise, they start the entire training from the beginning (from the Intro element).</p><p>In this case the user can try the test twice. If they do not pass the test in two attempts, they must restart the module from the beginning (from the Intro element).</p> <img src="/files/docs/2022/06/FAQ-Pass_Fail-settings.png" alt="" width="385" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;"> <img src="/files/docs/2022/06/FAQ-Pass_Fail-flow.png" alt="" width="718" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;">'
