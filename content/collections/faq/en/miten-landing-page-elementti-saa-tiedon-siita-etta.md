title: 'How does the Landing Page element receive information that a course module linked to it has been completed?'
answer: "<p>The module containing the Landing Page element must have some form of login method so that it can identify which course modules the person logged into the module containing the Landing Page element has completed. The identifier used for the participant is their email address. The login method can be, for example, SSO, Consumer Login or a Login element.</p><p>The Landing Page element defines the course modules to be completed, i.e. in practice links the courses to the Landing Page element. In the Landing Page element, participants can view the courses they have completed. In order to identify which course modules a user has completed, the linked course modules must also have some form of login method in addition to the Landing Page element, through which the participant''s email address used as their identifier can be obtained.</p><p>If the email address used for the participant''s login in the module containing the Landing Page element is the same as in the linked course module, the system interprets the same user as having completed the course. Based on this information, the Landing Page element informs the participant that they have completed the training.</p>"
