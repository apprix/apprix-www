title: 'Are there any limitations in Builder regarding how large a video or how many videos can be added to a single module?'
answer: '<p>The maximum size for videos uploaded to Builder is 100MB.</p><p>There is no limit on the number of videos used in a training, but a SCORM package can only be created from the training if the total size of files to be included in the package (incl. videos) does not exceed 200 MB.</p>'
