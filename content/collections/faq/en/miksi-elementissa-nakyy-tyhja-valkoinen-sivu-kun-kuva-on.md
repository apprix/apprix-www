title: 'Why does the element show a blank white page when an image has been added as a link?'
answer: '<p>If the image cannot be found at the link address or for some reason the image takes an unexpectedly long time to load (e.g. slow network connection), a blank white view will be shown in the element for some time. We recommend uploading image files to the image library and adding images to elements through it, rather than using links.</p>'
