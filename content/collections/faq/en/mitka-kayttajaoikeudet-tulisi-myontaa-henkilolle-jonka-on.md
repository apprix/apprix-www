title: 'What user permissions should be granted to a person who only needs to view the statistics of a specific module?'
answer: '<p>1. Open the Administrate view.</p><p>2. Select the user whose permissions you want to edit.</p><p>3. Select the Settings... button under the User''s Module Edit & Stats Permissions heading.</p><p>4. In the User''s Module Edit and Stats Permissions pop-up window, select the View option in the Access column for the module and tick the box in the Statistics column.</p><p>Note! If the None option is selected, it is not possible to tick the Statistics box, as selecting it is then disabled.</p> <img src="/files/docs/2020/07/userspermissions.png" alt="" width="268" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;"> <img src="/files/docs/2020/07/permissions.png" alt="" width="483" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;">'
