title: 'How do I unlink a submodule from the main module?'
answer: '<p>If you need to remove the link between a submodule and the main module, perform the following steps.</p><ul><li>Select the main module and click the Edit button. The Edit Module window opens.</li><li>Activate the submodule whose link you want to remove by clicking the submodule name in the pink Browse submodules list.</li><li>The submodule name appears at the top of the view below the main module name. At this point the submodule''s link from the main module can be removed. Note! The submodule must be in an unpublished state. Unpublish the submodule if necessary.</li></ul><p>4. Click the Submodule Settings button. Two options are visible: Change Linking and Change to Normal Module.</p><p>If you select Change Linking, the Change Linking pop-up window opens. The list shows all modules that contain a Submenu Buttons element (main modules).</p><p>5. Click the name of the module to which you want to link the submodule. The Submenu Settings pop-up window of the selected Submenu Buttons element opens.</p><p>In the view shown in the image above, you can choose how you want to link the submodule to the current main module: for example, by adding a new button to the menu (new submodule), naming it and linking the submodule to it from the pink Select button, or by linking the submodule to an existing button by clicking Select for that field.</p><p>If you do not want to link the submodule at all, you can select Not now, link to a menu later from the bottom right corner of the Change Linking pop-up window. In this case the submodule becomes an orphan (not linked to any submodule structure, and also not usable as a normal module).</p><p>You can later choose from the Submodule Settings menu if you want to link the submodule again to the main module of your choice, change it to a normal module, or delete it entirely by clicking the bin icon.</p> <img src="/files/docs/2020/07/unlink1.png" alt="" width="380" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;"> <img src="/files/docs/2020/07/unlink2.png" alt="" width="361" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;"> <img src="/files/docs/2020/07/unlink3-1000x689.png" alt="" width="386" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;"> <img src="/files/docs/2020/07/unlink4.png" alt="" width="426" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;">'
