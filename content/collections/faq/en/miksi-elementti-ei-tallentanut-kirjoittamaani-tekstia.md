title: 'Why did the element not save the text I typed?'
answer: '<p>If the mouse cursor is active in the text field (cursor is blinking) and you close the element, the text will not be saved and will therefore not appear in the preview (Preview) either.</p><p>To save the text, click outside the text field you are editing so the cursor is no longer active.</p><p>See the following example images:</p><p>The text field on the right is empty in the Preview mode because the cursor was active in the text field in the element''s Edit mode when the element was closed.</p> <img src="/files/docs/2020/07/textediting.png" alt="" width="653" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;"> <img src="/files/docs/2020/07/previwmode-1000x215.png" alt="" width="651" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;">'
