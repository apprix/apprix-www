title: 'Why is a submodule not visible in the submodule menu in preview (Preview) or in the published mode of the module, and I cannot navigate to the next submodule?'
answer: '<p>If the user has created a submodule and it is first in the list, but this submodule has not been published, it will not appear in preview or published mode. If it is set that submodules must be completed in order (Submodules have to be completed in the order defined below: YES), Builder will not allow the user to open the next submodule in preview or published mode, even if it is visible.</p><p>Publish the first submodule in the list to view the next submodule:</p><p>or set the Submodules have to be completed in the order defined below setting to: NO by first opening the Submenu Buttons element and clicking Edit Submodule Menu Settings from the orange gear icon at the top of the element''s edit view. Please note, however, that if this setting is turned off (NO) when the link is shared with course participants, participants can complete the submodules in any order they choose.</p> <img src="/files/docs/2021/09/Ensimmainen_submoduuli.jpg" alt="" width="435" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;"> <img src="/files/docs/2021/09/Submenu_Setting_YES.jpg" alt="" width="336" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;"> <img src="/files/docs/2021/09/Submenu_Sertting_NO.jpg" alt="" width="336" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;">'
