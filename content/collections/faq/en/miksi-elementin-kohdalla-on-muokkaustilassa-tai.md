title: 'Why is there a blank or partially blank page, or several consecutive blank pages, at an element in edit mode or preview, even though content was previously added to the element(s)?'
answer: '<p>This is a rare issue. Two or more users have possibly had the same module or submodule open in edit mode simultaneously, or some other temporary error has occurred (e.g. a network interruption).</p><p>The problem is easily resolved:</p><ul><li>Move the first blank/partially blank element in the module to another position in the module and save the module.</li><li>Then restore the element to its correct position and save the module again.</li></ul>'
