title: 'Can I use a login method (such as SSO, Tax Number Check, Email Registration) for a training that multiple people complete on the same device?'
answer: "<p>Using a login method such as SSO is not suitable for training on the same device, because after the first login Builder remembers the logged-in user and allows access to the training on subsequent attempts without asking for login credentials. For example, if the first user has logged in to the training with their SSO credentials and then a second user opens the same training, the first user''s details are recorded in the statistics, because the session is still using the first user''s credentials.</p>"
