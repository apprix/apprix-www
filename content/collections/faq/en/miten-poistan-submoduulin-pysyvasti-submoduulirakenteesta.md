title: 'How do I permanently delete a submodule from the submodule structure?'
answer: '<p>If you need to permanently remove a submodule linked to the main module in a submodule structure, perform the following steps.</p><ul><li>Select the main module and click the Edit button. The Edit Module window opens.</li></ul><p>2. Activate the submodule you want to remove by clicking the submodule name in the Browse submodules list.</p><p>The submodule name appears at the top of the view below the main module name. At this point the submodule is ready to be removed.</p><p>3. Click the module name below the main module and then the bin icon in the top right corner of the view.</p><p>Builder asks whether you really want to remove the submodule.</p><p>4. Select the OK button. The submodule will be removed from the submodule structure and permanently deleted.</p><p>Note! Remember to also remove the submodule''s menu button from the Submenu Buttons element if the button is no longer needed.</p> <img src="/files/docs/2020/07/remove1.png" alt="" width="428" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;"> <img src="/files/docs/2020/07/remove2.png" alt="" width="398" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;"> <img src="/files/docs/2020/07/remove3.png" alt="" width="383" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;"> <img src="/files/docs/2020/07/remove4-1000x656.png" alt="" width="408" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;">'
