answer: '<p>In preview mode, module statistics are not recorded. For this reason, the "Pass/Fail" and "Failure" elements do not work in preview mode either. Scores are only calculated in Published mode.</p><p>Why does Builder work this way? Because the statistics record participants'''' completion data, not module testing data.</p>'
