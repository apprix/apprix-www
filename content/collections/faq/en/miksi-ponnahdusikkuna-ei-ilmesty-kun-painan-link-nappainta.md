title: 'Why does the pop-up window not appear when I press the Link button?'
answer: "<p>The pop-up window does not appear because the browser is blocking it. This may be due to the browser''s security settings, or the browser may temporarily block pop-up windows if the \"Do not allow this page to create new windows / Block this page from creating new windows\" setting is selected. This setting temporarily prevents pop-up windows from opening. Refreshing the page in Firefox and reopening the browser in Internet Explorer should reset the setting.</p><p>You can also try using a different browser if the pop-up window does not appear.</p><p>If the window still does not appear, it is advisable to contact your organisation''s IT support. They may know if your organisation''s browsers have security settings that block pop-up windows.</p>"
