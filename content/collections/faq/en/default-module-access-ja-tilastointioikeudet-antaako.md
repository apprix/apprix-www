title: 'Default Module Access and statistics permissions - Does Default Module Access = Edit also grant statistics permissions?'
answer: '"<p>✅ Yes it does! When a user is set to Default Module Access = Edit, they gain the right to:</p><ul><li>edit all existing and future modules</li><li>view module statistics</li></ul><p>Meaning of Default Module Access settings</p><p>Setting Permissions</p><p>None User can only see modules they have created themselves</p><p>View (System Default) User can preview and copy all modules, but cannot edit them or view statistics</p><p>Edit User can edit and view statistics for all current and future modules</p><p>⚠️ Note: Module-specific permissions can override the default setting</p><p>Even if a facilitator has been given Default Module Access = Edit, their permissions may still be restricted if module-specific permissions have previously been defined for them in the User''s Module Edit & Stats Permissions window.</p><p>Important: ➡️ Module-specific settings (View/Edit) override the Default Module Access setting for those particular modules.</p><p>Example situation</p><p>Situation:</p><ul><li>The facilitator has been set to Default Module Access = Edit</li><li>However, in the User''s Module Edit & Stats Permissions window, two modules have been granted View access</li></ul><p>Consequences:</p><ul><li>The facilitator can edit all other modules</li><li>For these two modules, they can only:</li><li>view (not edit)</li><li>copy the content using the Duplicate button</li><li>cannot view statistics</li><li>These modules were created by another facilitator</li></ul><p>Descriptions (image references)</p><p>Image 1 – Anna Editor''s Default Module Access = Edit. However, two modules have been set to Access = View. Anna can view and copy them, but cannot edit them or view statistics.</p><p>Image 2 – In the Builder view, Anna can edit all other modules except those that have been separately granted View access.</p>" <img src="/files/docs/2025/07/Default-module-access.jpg" alt="" width="2229" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;"> <img src="/files/docs/2025/07/Default-module-access-2.jpg" alt="" width="2007" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;">'
