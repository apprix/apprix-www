title: 'Commenting tool operation and error situations - How is the commenting tool designed?'
answer: '<p>The commenting tool is designed so that multiple users can comment on the same element simultaneously.</p><p>If comments do not appear in acknowledged or deleted comments, they have not been saved. In this case the tool should display a red error message:</p><p>"Could not save comment most likely due to a network error, please try again in a few moments."</p><p>Possible causes of the save problem:</p><ul><li>Network connection disruption</li><li>Other technical issues or intermittent server connection</li></ul><p>⚠️ Note! Comments that have not been saved cannot be recovered later.</p>'
