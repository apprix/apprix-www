title: 'What happens to module completion data if I delete an element or an item from an element?'
answer: '<p>Completions are not removed. Changing the module structure causes the following in statistics:</p><ul><li>"Progress" will appear the same for old completions as before. New completions will then have progress according to the module''''s new structure. For example, if the module previously had 10 elements and one element is removed, old completions that were finished will still show progress = 10. For new completions it will be 9. The same applies to the points collected from the module if they are changed.</li><li>Deleting a task element (e.g. checkbox-two): deleted task elements are archived and are not shown in statistics by default. They can, however, be made visible with the "Show archived" button</li><li>Deleting a content element (e.g. info-fullbg): no effect on stats other than new completions having a different progress than old ones.</li></ul>'
