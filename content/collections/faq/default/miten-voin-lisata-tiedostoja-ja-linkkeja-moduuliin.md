answer: 'Seuraamalla näitä ohjeita voit lisätä moduuliin linkkejä ja tiedostoja, jotka ovat suorittajan saatavilla koko moduulin suorittamisen ajan: <b>Hampurilaisvalikko</b> Hampurilaisvalikon painike sijaitsee elementtien oikeassa yläkulmassa. Klikkaamalla valikkoa aukeaa lista tiedostoja ja linkkejä, jotka ovat koulutuksen suorittajien käytössä. <img src="/files/docs/2020/11/hamburger1-1000x353.png" alt="" width="489" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;"> <b>Muokkausohjeet</b> Klikatessasi hampurilaisvalikko aukeaa <b>Link menu settings</b><span style="font-weight: 400;"> ponnahdusikkuna. Tässä ikkunassa voi lisätä <strong>linkkejä</strong> (https://www.google.com/) ja <strong>tiedostoja</strong> (pdf, doc/docx, xls/xlsx, ppt/pptx, txt/csv, zip) hampurilaisvalikkoon.</span> <b>Huom!</b> Linkit ja tiedostot, jotka <ul> <li style="font-weight: 400;"><span style="font-weight: 400;">lisätään tai poistetaan minkä tahansa elementin hampurilaisvalikon kautta päivittyvät automaattisesti jokaiseen moduulin elementtiin.</span></li> <li style="font-weight: 400;"><span style="font-weight: 400;">lisätään päämoduuliin eivät päivity automaattisesti submoduuleihin tai submoduulista päämoduuliin ja muihin submoduuleihin, sillä hampurilaisvalikko on moduulikohtainen.</span></li> </ul> <img src="/files/docs/2020/11/hamburger2.png" alt="" width="454" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;"> <b>+ ADD LINK -painike</b> Klikkaa <b>+ ADD LINK </b>-painiketta, kun haluat lisätä linkkejä hampurilaisvalikkoon. <b>Browse Media Library</b><span style="font-weight: 400;"> -ponnahdusikkuna aukeaa. Tässä ikkunassa näet listan aikaisemmin lisätyistä linkeistä, jos niitä on. Jos etsimäsi linkki ei löydy ikkunan listasta vielä, klikkaa oranssia <b>Add new link</b>…-painiketta. Aukeaa uusi ponnahdusikkuna, jossa pyydetään antamaan nimi linkille. Tämä nimi näkyy linkin nimenä ikkunan listassa. Syötä nimi linkille (esim. google.com) ja klikkaa <strong>OK</strong>. Aukeaa jälleen uusi ponnahdusikkuna, jossa kysytään, halutaanko linkki säilyttää moduulin linkkikirjastossa, jolloin kyseinen linkki näkyy listättyjen linkkien listassa. Syötä linkin nimi (esim. google.com) ja klikkaa <strong>OK</strong>. Ponnahdusikkuna ja <b>Browse Media Library </b>-ikkuna sulkeutuvat.</span> <img src="/files/docs/2020/11/hamburger3.png" alt="" width="394" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;"> <span style="font-weight: 400;">Klikkaa uudelleen <b>+</b> <b>ADD LINK </b>-painiketta <b>Link Menu Settings </b>-valikossa. Valitse <b>Browse Media Library </b>-ponnahdusikkunassa haluamasi linkki ja klikkaa <b>Insert </b></span><span style="font-weight: 400;"><b>Selected</b> </span><span style="font-weight: 400;">-painiketta ikkunan oikeassa alalaidassa. Linkki ilmestyy nyt <b>Link Menu Settings </b>-ikkunaan. </span> <img src="/files/docs/2020/11/hamburger4-1000x595.png" alt="" width="442" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;"> Valitse <strong>Done</strong>, jos sinun et halua lisätä lisää linkkejä. Linkki on nyt saatavilla <strong>hampurilaisvalikosta</strong>. <b>+ ADD FILE -painike</b> Klikkaa <b>+</b> <b>ADD FILE </b>-painiketta, kun sinun tarvitsee lisätä tiedosto saataville hampurilaisvalikkoon. <b>Browse Media Library </b>-ponnahdusikkuna aukeaa. <span style="font-weight: 400;">Tässä ikkunassa näet listan aikaisemmin lisätyistä tiedostoista, jos niitä on. Jos etsimäsi tiedosto ei vielä löydy ikkunan listasta, klikkaa oranssia <b>Add new file</b>… -painiketta.</span> Huom! Rastita ruutu <b>Show uploaded file in library</b>, jos haluat tiedoston näkyvän ikkunan tiedostolistauksessa eli tiedostokirjastossa. <img src="/files/docs/2020/11/hamburger5.png" alt="" width="411" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;"> <strong>File Explorer</strong> -ikkuna aukeaa. Valitse tiedosto, jonka haluat lisätä kirjastoon. Aukeaa uusi ponnahdusikkuna, jossa pyydetään antamaan tiedostolle nimi, jolla tiedoston löytää kirjastosta. Syötä tiedostolle nimi ja paina <strong>OK</strong>. Ponnahdusikkuna ja <b>Browse Media Library </b>-ikkuna sulkeutuvat. Klikkaa jälleen <b>+</b> <b>ADD FILE </b>-painiketta <b>Link Menu Settings </b>-ikkunassa. Valitse tiedosto <b>Browse Media Library </b>-ikkunassa ja klikkaa <b>Insert Selected </b>-painiketta ikkunan oikeassa alalaidassa. Tiedosto ilmestyy nyt <b>Link Menu Settings </b>-ikkunaan. Valitse <strong>Done</strong>, jos et halua lisätä muita tiedostoja. Tiedosto on nyt lisätty nähtäville <strong>hampurilaisvalikkoon.</strong> <img src="/files/docs/2020/11/hamburger6-1000x589.png" alt="" width="416" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;"> <b>Link Menu Settings -valikkoon </b>lisätyt tiedostot ja linkit ovat nyt tarkasteltavissa kaikista moduulin elementeistä <strong>hampurilaisvalikosta</strong>. Jos haluat poistaa tiedoston tai linkin, klikkaa<strong> roskakoria</strong> linkin/tiedoston nimen oikealla puolella. <img src="/files/docs/2020/11/hamburger7.png" alt="" width="357" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;"> Jos haluat poistaa linkin ja tiedoston näkyvistä <strong>Media Library </strong>-ikkunan listasta, klikkaa <b>REMOVE SELECTED FROM LIBRARY</b> -painiketta ikkunan vasemmassa alalaidassa. Huomioithan, että linkki/tiedosto poistetaan tällöin vain <b>Media Library </b>-ikkunassa näkyvästä listasta, mutta ei moduulista, jos linkki tai tiedosto on lisättynä moduulin elementin <b>Link Menu Settings</b> -ponnahdusikkunasta. <img src="/files/docs/2020/11/hamburger8.png" alt="" width="343" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;">'
