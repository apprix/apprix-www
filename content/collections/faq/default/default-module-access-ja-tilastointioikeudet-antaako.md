title: 'Default Module Access ja tilastointioikeudet - Antaako Default Module Access = Edit myös tilastointioikeudet?'
category: 'Käyttäjäoikeuksien hallinta'
order: 8
answer: '<p>✅ Kyllä antaa! Kun käyttäjälle asetetaan Default Module Access = Edit, hän saa oikeuden:</p><ul><li>muokata kaikkia olemassa olevia ja tulevia moduuleja</li><li>tarkastella moduulien tilastoja</li></ul><p>Default Module Access -asetusten merkitys</p><p>Asetus Oikeudet</p><p>None Käyttäjä näkee vain itse luomansa moduulit</p><p>View (System Default) Käyttäjä voi esikatsella ja kopioida kaikkia moduuleja, mutta ei muokata tai nähdä tilastoja</p><p>Edit Käyttäjä voi muokata ja tarkastella tilastoja kaikista nykyisistä ja tulevista moduuleista</p><p>⚠️ Huomio: Moduulikohtaiset oikeudet voivat ohittaa oletusasetuksen</p><p>Vaikka fasilitaattorille olisi annettu Default Module Access = Edit, hänen oikeutensa voivat silti olla rajoitetut, jos hänelle on aiemmin määritetty moduulikohtaisia oikeuksia User’s Module Edit & Stats Permissions -ikkunassa.</p><p>Tärkeää: ➡️ Moduulikohtaiset asetukset (View/Edit) korvaavat Default Module Access -asetuksen kyseisten moduulien osalta.</p><p>Esimerkki tilanteesta</p><p>Tilanne:</p><ul><li>Fasilitaattorille on asetettu Default Module Access = Edit</li><li>Kuitenkin User’s Module Edit & Stats Permissions -ikkunassa kahdelle moduulille on annettu oikeudeksi View</li></ul><p>Seuraukset:</p><ul><li>Fasilitaattori voi muokata kaikkia muita moduuleja</li><li>Näitä kahta moduulia hän voi vain:</li><li>tarkastella (ei muokata)</li><li>kopioida sisällön Duplicate-painikkeella</li><li>ei voi tarkastella tilastoja</li><li>Nämä moduulit on luonut toinen fasilitaattori</li></ul><p>️ Kuvaukset (viittaukset kuviin)</p><p>Kuva 1 – Anna Editorin Default Module Access = Edit. Kaksi moduulia on kuitenkin asetettu tilaan Access = View. Anna voi tarkastella ja kopioida ne, mutta ei muokata tai nähdä tilastoja.</p><p>Kuva 2 – Builder-näkymässä Anna voi muokata kaikkia muita moduuleja, paitsi niitä, joille on erikseen annettu View-oikeus.</p>'
