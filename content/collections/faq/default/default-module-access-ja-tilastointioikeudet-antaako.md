answer: '<strong>✅ Kyllä antaa!</strong> Kun käyttäjälle asetetaan <strong>Default Module Access = Edit</strong>, hän saa oikeuden: <ul> <li>muokata kaikkia olemassa olevia ja tulevia moduuleja</li> <li>tarkastella moduulien tilastoja</li> </ul> <strong>Default Module Access -asetusten merkitys</strong> <table> <thead> <tr> <td><strong>Asetus</strong></td> <td><strong>Oikeudet</strong></td> </tr> </thead> <tbody> <tr> <td><strong>None</strong></td> <td>Käyttäjä näkee vain itse luomansa moduulit</td> </tr> <tr> <td><strong>View (System Default)</strong></td> <td>Käyttäjä voi esikatsella ja kopioida kaikkia moduuleja, mutta ei muokata tai nähdä tilastoja</td> </tr> <tr> <td><strong>Edit</strong></td> <td>Käyttäjä voi muokata ja tarkastella tilastoja kaikista nykyisistä ja tulevista moduuleista</td> </tr> </tbody> </table> ⚠️ <strong>Huomio</strong>: Moduulikohtaiset oikeudet voivat ohittaa oletusasetuksen Vaikka fasilitaattorille olisi annettu <strong>Default Module Access = Edit</strong>, hänen oikeutensa voivat silti olla rajoitetut, jos hänelle on aiemmin määritetty moduulikohtaisia oikeuksia <strong>User’s Module Edit &amp; Stats Permissions</strong> -ikkunassa. <strong>Tärkeää:</strong> ➡️ Moduulikohtaiset asetukset (View/Edit) <strong>korvaavat</strong> Default Module Access -asetuksen kyseisten moduulien osalta. <strong>Esimerkki tilanteesta</strong> <strong>Tilanne:</strong> <ul> <li>Fasilitaattorille on asetettu <strong>Default Module Access = Edit</strong></li> <li>Kuitenkin <strong>User’s Module Edit &amp; Stats Permissions</strong> -ikkunassa kahdelle moduulille on annettu oikeudeksi <strong>View</strong></li> </ul> <strong>Seuraukset:</strong> <ul> <li>Fasilitaattori voi muokata kaikkia muita moduuleja</li> <li>Näitä kahta moduulia hän voi vain: <ul> <li>tarkastella (ei muokata)</li> <li>kopioida sisällön <strong>Duplicate</strong>-painikkeella</li> <li><strong>ei voi tarkastella tilastoja</strong></li> </ul> </li> <li>Nämä moduulit on luonut toinen fasilitaattori</li> </ul> ️ <strong>Kuvaukset (viittaukset kuviin)</strong> <strong>Kuva 1</strong> – Anna Editorin <strong>Default Module Access = Edit</strong>. Kaksi moduulia on kuitenkin asetettu tilaan <strong>Access = View</strong>. Anna voi tarkastella ja kopioida ne, mutta ei muokata tai nähdä tilastoja. <img src="/files/docs/2025/07/Default-module-access.jpg" alt="" width="2229" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;"> <strong>Kuva 2</strong> – Builder-näkymässä Anna voi muokata kaikkia muita moduuleja, paitsi niitä, joille on erikseen annettu <strong>View</strong>-oikeus. <img src="/files/docs/2025/07/Default-module-access-2.jpg" alt="" width="2007" style="max-width:100%;height:auto;display:block;margin:0.75rem 0;">'
