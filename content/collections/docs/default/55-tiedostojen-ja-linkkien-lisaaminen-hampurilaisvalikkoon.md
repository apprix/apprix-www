Seuraamalla näitä ohjeita voit lisätä moduuliin linkkejä ja tiedostoja, jotka ovat suorittajan saatavilla koko moduulin suorittamisen ajan.
<h4 class="fusion-responsive-typography-calculated" data-fontsize="18" data-lineheight="27px">Mikä on hampurilaisvalikko?</h4>
<strong>Hampurilaisvalikon</strong> painike sijaitsee elementtien oikeassa yläkulmassa. Klikkaamalla valikkoa aukeaa lista tiedostoja ja linkkejä, jotka ovat koulutuksen suorittajien käytössä.
<h4 class="fusion-responsive-typography-calculated" data-fontsize="18" data-lineheight="27px"><b>Muokkausohjeet</b></h4>
Klikatessasi hampurilaisvalikko aukeaa <b>Link menu settings</b> ponnahdusikkuna. Tässä ikkunassa voi lisätä <strong>linkkejä</strong> (https://www.google.com/) ja <strong>tiedostoja</strong> (pdf, doc/docx, xls/xlsx, ppt/pptx, txt/csv, zip) hampurilaisvalikkoon.

<b>Huom!</b>

Linkit ja tiedostot, jotka
<ul>
 <li>lisätään tai poistetaan minkä tahansa elementin hampurilaisvalikon kautta päivittyvät automaattisesti jokaiseen moduulin elementtiin.</li>
 <li>lisätään päämoduuliin eivät päivity automaattisesti submoduuleihin tai submoduulista päämoduuliin ja muihin submoduuleihin, sillä hampurilaisvalikko on moduulikohtainen.</li>
</ul>
<h4>+ ADD LINK -painike</h4>
<ul>
 <li>Klikkaa <b>+ ADD LINK </b>-painiketta, kun haluat lisätä linkkejä hampurilaisvalikkoon. <b>Browse Media Library</b> -ponnahdusikkuna aukeaa. Tässä ikkunassa näet listan aikaisemmin lisätyistä linkeistä, jos niitä on. </li>
 <li>Jos etsimäsi linkki ei löydy ikkunan listasta vielä, klikkaa oranssia <b>Add new link</b>…-painiketta. Aukeaa uusi ponnahdusikkuna, jossa pyydetään antamaan nimi linkille. Tämä nimi näkyy linkin nimenä ikkunan listassa. Syötä nimi linkille (esim. google.com) ja klikkaa <strong>OK</strong>. </li>
 <li>Aukeaa jälleen uusi ponnahdusikkuna, jossa kysytään, halutaanko linkki säilyttää moduulin linkkikirjastossa, jolloin kyseinen linkki näkyy listättyjen linkkien listassa. Syötä linkin nimi (esim. google.com) ja klikkaa <strong>OK</strong>. Ponnahdusikkuna ja <b>Browse Media Library </b>-ikkuna sulkeutuvat.</li>
 <li>Klikkaa uudelleen <b>+</b> <b>ADD LINK </b>-painiketta <b>Link Menu Settings </b>-valikossa. Valitse <b>Browse Media Library </b>-ponnahdusikkunassa haluamasi linkki ja klikkaa <b>Insert </b><b>Selected</b> -painiketta ikkunan oikeassa alalaidassa. Linkki ilmestyy nyt <b>Link Menu Settings </b>-ikkunaan. </li>
 <li>Valitse <strong>Done</strong>, jos sinun et halua lisätä lisää linkkejä. Linkki on nyt saatavilla <strong>hampurilaisvalikosta</strong>.</li>
</ul>
<h4>+ ADD FILE -painike</h4>
<ul>
 <li>Klikkaa <b>+</b> <b>ADD FILE </b>-painiketta, kun sinun tarvitsee lisätä tiedosto saataville hampurilaisvalikkoon. <b>Browse Media Library </b>-ponnahdusikkuna aukeaa. Tässä ikkunassa näet listan aikaisemmin lisätyistä tiedostoista, jos niitä on. 
<ul>
 <li>Jos etsimäsi tiedosto ei vielä löydy ikkunan listasta, klikkaa oranssia <b>Add new file</b>… -painiketta.</li>
</ul>
</li>
 <li>Huom! Rastita ruutu <b>Show uploaded file in library</b>, jos haluat tiedoston näkyvän ikkunan tiedostolistauksessa eli tiedostokirjastossa.</li>
 <li><strong>File Explorer</strong> -ikkuna aukeaa. Valitse tiedosto, jonka haluat lisätä kirjastoon. Aukeaa uusi ponnahdusikkuna, jossa pyydetään antamaan tiedostolle nimi, jolla tiedoston löytää kirjastosta. Syötä tiedostolle nimi ja paina <strong>OK</strong>. Ponnahdusikkuna ja <b>Browse Media Library </b>-ikkuna sulkeutuvat.</li>
 <li>Klikkaa jälleen <b>+</b> <b>ADD FILE </b>-painiketta <b>Link Menu Settings </b>-ikkunassa. Valitse tiedosto <b>Browse Media Library </b>-ikkunassa ja klikkaa <b>Insert Selected </b>-painiketta ikkunan oikeassa alalaidassa. Tiedosto ilmestyy nyt <b>Link Menu Settings </b>-ikkunaan.</li>
 <li>Valitse <strong>Done</strong>, jos et halua lisätä muita tiedostoja. Tiedosto on nyt lisätty nähtäville <strong>hampurilaisvalikkoon.</strong></li>
</ul>
<b>Link Menu Settings -valikkoon </b>lisätyt tiedostot ja linkit ovat nyt tarkasteltavissa kaikista moduulin elementeistä <strong>hampurilaisvalikosta</strong>. Jos haluat poistaa tiedoston tai linkin, klikkaa<strong> roskakoria</strong> linkin/tiedoston nimen oikealla puolella.

Jos haluat poistaa linkin ja tiedoston näkyvistä <strong>Media Library </strong>-ikkunan listasta, klikkaa <b>REMOVE SELECTED FROM LIBRARY</b> -painiketta ikkunan vasemmassa alalaidassa. Huomioithan, että linkki/tiedosto poistetaan tällöin vain <b>Media Library </b>-ikkunassa näkyvästä listasta, mutta ei moduulista, jos linkki tai tiedosto on lisättynä moduulin elementin <b>Link Menu Settings</b> -ponnahdusikkunasta.
