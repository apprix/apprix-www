<p>Seuraamalla näitä ohjeita voit lisätä moduuliin linkkejä ja tiedostoja, jotka ovat suorittajan saatavilla koko moduulin suorittamisen ajan.</p><h3>Mikä on hampurilaisvalikko?</h3><p><strong>Hampurilaisvalikon</strong> painike sijaitsee elementtien oikeassa yläkulmassa. Klikkaamalla valikkoa aukeaa lista tiedostoja ja linkkejä, jotka ovat koulutuksen suorittajien käytössä.</p><h3>Muokkausohjeet</h3><p>Klikatessasi hampurilaisvalikko aukeaa <strong>Link menu settings</strong> ponnahdusikkuna. Tässä ikkunassa voi lisätä <strong>linkkejä</strong> (https://www.google.com/) ja <strong>tiedostoja</strong> (pdf, doc/docx, xls/xlsx, ppt/pptx, txt/csv, zip) hampurilaisvalikkoon.</p><p><strong>Huom!</strong></p><p>Linkit ja tiedostot, jotka</p><ul><li><p>lisätään tai poistetaan minkä tahansa elementin hampurilaisvalikon kautta päivittyvät automaattisesti jokaiseen moduulin elementtiin.</p></li><li><p>lisätään päämoduuliin eivät päivity automaattisesti submoduuleihin tai submoduulista päämoduuliin ja muihin submoduuleihin, sillä hampurilaisvalikko on moduulikohtainen.</p></li></ul><h3>+ ADD LINK -painike</h3><ul><li><p>Klikkaa <strong>+ ADD LINK </strong>-painiketta, kun haluat lisätä linkkejä hampurilaisvalikkoon. <strong>Browse Media Library</strong> -ponnahdusikkuna aukeaa. Tässä ikkunassa näet listan aikaisemmin lisätyistä linkeistä, jos niitä on.</p></li><li><p>Jos etsimäsi linkki ei löydy ikkunan listasta vielä, klikkaa oranssia <strong>Add new link</strong>…-painiketta. Aukeaa uusi ponnahdusikkuna, jossa pyydetään antamaan nimi linkille. Tämä nimi näkyy linkin nimenä ikkunan listassa. Syötä nimi linkille (esim. google.com) ja klikkaa <strong>OK</strong>.</p></li><li><p>Aukeaa jälleen uusi ponnahdusikkuna, jossa kysytään, halutaanko linkki säilyttää moduulin linkkikirjastossa, jolloin kyseinen linkki näkyy listättyjen linkkien listassa. Syötä linkin nimi (esim. google.com) ja klikkaa <strong>OK</strong>. Ponnahdusikkuna ja <strong>Browse Media Library </strong>-ikkuna sulkeutuvat.</p></li><li><p>Klikkaa uudelleen <strong>+</strong> <strong>ADD LINK </strong>-painiketta <strong>Link Menu Settings </strong>-valikossa. Valitse <strong>Browse Media Library </strong>-ponnahdusikkunassa haluamasi linkki ja klikkaa <strong>Insert Selected</strong> -painiketta ikkunan oikeassa alalaidassa. Linkki ilmestyy nyt <strong>Link Menu Settings </strong>-ikkunaan.</p></li><li><p>Valitse <strong>Done</strong>, jos sinun et halua lisätä lisää linkkejä. Linkki on nyt saatavilla <strong>hampurilaisvalikosta</strong>.</p></li></ul><h3>Miten lisäät linkkejä hampurilaisvalikkoon?</h3><p><img src="statamic://asset::files::docs/2024/07/Hamburger-link.gif"></p><h3>+ ADD FILE -painike</h3><ul><li><p>Klikkaa <strong>+</strong> <strong>ADD FILE </strong>-painiketta, kun sinun tarvitsee lisätä tiedosto saataville hampurilaisvalikkoon. <strong>Browse Media Library </strong>-ponnahdusikkuna aukeaa. Tässä ikkunassa näet listan aikaisemmin lisätyistä tiedostoista, jos niitä on.</p><ul><li><p>Jos etsimäsi tiedosto ei vielä löydy ikkunan listasta, klikkaa oranssia <strong>Add new file</strong>… -painiketta.</p></li></ul></li><li><p>Huom! Rastita ruutu <strong>Show uploaded file in library</strong>, jos haluat tiedoston näkyvän ikkunan tiedostolistauksessa eli tiedostokirjastossa.</p></li><li><p><strong>File Explorer</strong> -ikkuna aukeaa. Valitse tiedosto, jonka haluat lisätä kirjastoon. Aukeaa uusi ponnahdusikkuna, jossa pyydetään antamaan tiedostolle nimi, jolla tiedoston löytää kirjastosta. Syötä tiedostolle nimi ja paina <strong>OK</strong>. Ponnahdusikkuna ja <strong>Browse Media Library </strong>-ikkuna sulkeutuvat.</p></li><li><p>Klikkaa jälleen <strong>+</strong> <strong>ADD FILE </strong>-painiketta <strong>Link Menu Settings </strong>-ikkunassa. Valitse tiedosto <strong>Browse Media Library </strong>-ikkunassa ja klikkaa <strong>Insert Selected </strong>-painiketta ikkunan oikeassa alalaidassa. Tiedosto ilmestyy nyt <strong>Link Menu Settings </strong>-ikkunaan.</p></li><li><p>Valitse <strong>Done</strong>, jos et halua lisätä muita tiedostoja. Tiedosto on nyt lisätty nähtäville <strong>hampurilaisvalikkoon.</strong></p></li></ul><h3>Miten lisäät tiedostoja hampurilaisvalikkoon?</h3><p><img src="statamic://asset::files::docs/2024/07/Hamburger-file.gif"><strong>Link Menu Settings -valikkoon </strong>lisätyt tiedostot ja linkit ovat nyt tarkasteltavissa kaikista moduulin elementeistä <strong>hampurilaisvalikosta</strong>. Jos haluat poistaa tiedoston tai linkin, klikkaa<strong> roskakoria</strong> linkin/tiedoston nimen oikealla puolella.</p><p>Jos haluat poistaa linkin ja tiedoston näkyvistä <strong>Media Library </strong>-ikkunan listasta, klikkaa <strong>REMOVE SELECTED FROM LIBRARY</strong> -painiketta ikkunan vasemmassa alalaidassa. Huomioithan, että linkki/tiedosto poistetaan tällöin vain <strong>Media Library </strong>-ikkunassa näkyvästä listasta, mutta ei moduulista, jos linkki tai tiedosto on lisättynä moduulin elementin <strong>Link Menu Settings</strong> -ponnahdusikkunasta.</p>
