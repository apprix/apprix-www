<p><strong>Edit Points </strong>-ominaisuus löytyy moduulin editointinäkymän <strong>ratassymbolista </strong>avautuvasta alasvetovalikosta klikkaamalla <strong>Edit Points</strong>. Pisteiden lasku tapahtuu automaattisesti prosenttiyksiköittäin, eikä pisteitä tarvitse yleensä muuttaa luodessa uuden moduulin. Maksimitulos on 100% ja se jaetaan automaattisesti moduuliin lisättyjen tehtäväelementtien kesken.</p><p>Jos haluat, voit muuttaa tehtävien suhteellista tärkeyttä. Perusasetuksena jokainen tehtävä on 1 pisteen arvoinen eli yhtä tärkeä pisteytysarvoltansa. Voit tehdä tehtävästä pisteytykseltään tärkeämmän lisäämällä sen pistearvoa. Voit myös poistaa tehtävän pistearvon poistamalla tehtävän pisteet ja korvaamalla ne 0 pisteellä.</p><p><strong>Huom!</strong> <strong>Edit Points</strong> -näkymästä voi myös muokata moduulin prosentuaalista läpipääsyrajaa.</p><p>Jos moduulin loppupäässä on lopputesti sekä läpipääsyä säätelevä <a href="/help/730-passfail-and-passfail-always"><strong>Pass/Fail-elementti</strong></a> ja ainoastaan testin pisteet tulisi huomioida pisteenlaskussa, poista pisteet kaikista paitsi testiin kuuluvista elementeistä ja säädä prosentuaalinen läpipääsyraja halutulle tasolle.</p><h3>Miten muokkaat elementtipisteitä ja moduulin lopputestin läpäisyyn tarvittavien pisteiden prosenttiosuutta?</h3><p><img src="statamic://asset::files::docs/2024/08/Edit-Points.gif"></p>
