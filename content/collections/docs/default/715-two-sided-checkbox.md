title: '7.15 Two-Sided Checkbox'
---
<b>Two-Sided Checkbox</b> on tehtäväelementti ja sitä käytetään yleisesti testeissä <b>Pass / Fail</b> -elementin kanssa.

Suorittajan on valittava oikea väittämä kahdesta vaihtoehdosta. Tekstiruutu, jossa on lisätietoja, näytetään jokaisen valinnan jälkeen.
<h4>Elementin muokkausohjeet</h4>
<ul>
 <li aria-level="1">Kirjoita tehtävän esittely johdantolaatikkoon, jos haluat näyttää johdannon tehtävän alussa. Voit myös jättää laatikon tyhjäksi. Tällöin johdantolaatikko ei näy suorittajalle.</li>
 <li aria-level="1">Lisää taustakuva klikkaamalla <b>Change image</b> -kuvaketta elementin oikeassa yläkulmassa.</li>
 <li aria-level="1">Klikkaa elementin editointinäkymän yläosassa sijaitsevasta ratasvalikosta <b>Element settings</b> -vaihtoehtoa. </li>
 <li aria-level="1">Lisää haluamasi määrä kenttiä <b>Element settings</b> –ikkunasta. </li>
 <li aria-level="1">Muokkaa väittämät.</li>
</ul>
<b>“Answer Mode” -toiminnallisuus</b>

Jos haluat aktivoida <b>Answer Mode</b> -toiminnallisuuden, klikkaa rataskuvaketta ikkunan vasemmassa reunassa.

<b>Vaihtoehto 1: Answer Mode on kytketty pois päältä</b>
<ul>
 <li aria-level="1">Klikkaa kunkin kohteen valintaruutua. Vastaustekstin ponnahdusikkuna tulee näkyviin.</li>
 <li aria-level="1">Muokkaa sen sisältöä.</li>
 <li aria-level="1">Kirjoita tähän lisätietoa väittämän aiheesta. Huomaa, että sisältö on sama riippumatta annetusta vastauksesta (esim. Vasen/Oikea).</li>
</ul>
<b>Vaihtoehto 2: Answer Mode on kytketty päälle</b>
<ul>
 <li aria-level="1">Ota Answer Mode käyttöön klikkaamalla asetuspalkin sinistä painiketta.</li>
 <li aria-level="1">Klikkaa nyt kohteen valintaruutua, niin vastauksen teksti avautuu ponnahdusikkunassa. Ponnahdusikkunan yläosaan ilmestyvät OIKEA (Right) ja VÄÄRÄ (Wrong) -vaihtoehdot. Perustelee vastaukset eri vaihtoehdolle.</li>
</ul>
<b>HUOM!</b> Jos ponnahdusikkunan vastausteksti jätetään tyhjäksi, vastausteksti-ikkunaa ei tule näkyviin, kun suorittaja on valinnut vastausruudun.
