<p><strong>Kirjautuminen PIN-koodilla</strong></p>

<p>Tämä kirjautumistapa poistaa tarpeen muistaa salasanoja. Kurssin osallistuja syöttää vain perustietonsa (etunimi, sukunimi ja sähköpostiosoite) <strong>Tervetuloa-ikkunaan</strong> aina kurssin alussa.</p>

<p><strong>"Email One-Time PIN (OTP)" -tunnistautumismenetelmän aktivointi</strong></p>

<ol>
 <li>Avaa moduuli <strong>Muokkaustilassa (Edit module)</strong>.</li>
 <li>Valitse moduulin yläreunassa olevasta <strong>asetukset (hammasratas) -kuvakkeesta</strong> kohta <strong>User Authentication…</strong>. <em>User Authentication &amp; Access</em> -pudotusvalikko avautuu.</li>
 <li>Valitse <strong>Email One-Time PIN (OTP)</strong>. Valinta näkyy nyt hammasratas-pudotusvalikossa.</li>
</ol>

<p><strong>Kuinka se toimii</strong></p>

<ul>
 <li>Kurssin osallistuja syöttää perustiedot (sähköpostiosoite, etunimi, sukunimi) <strong>Tervetuloa-ikkunaan</strong> ja sitten napsauttaa Send Pin Code -painiketta. PIN-koodi luodaan automaattisesti tunnistautumisalustan toimesta.</li>
 <li>PIN-koodi lähetetään osallistujalle <strong>sähköpostitse</strong>.</li>
 <li>Osallistuja liittää PIN-koodin <strong>tunnistautumisen ponnahdusikkunaan</strong>.</li>
 <li>Kurssi on valmis alkamaan.</li>
</ul>

<h3>Huom! Käyttäjän sähköpostiosoitteen, etunimen ja sukunimen on oltava yksilöllisiä.</h3>

<p><img src="/files/docs/2025/12/pin-code-message-1.png"></p>

<p><strong>Tärkeät huomiot</strong></p>

<ul>
 <li>Uusi, ainutkertainen <strong>PIN-koodi</strong> lähetetään jokaisella kirjautumiskerralla.</li>
 <li>Koodi on syötettävä <strong>10 minuutin kuluessa</strong> vastaanottamisesta.</li>
 <li>Käyttäjän ei tarvitse tehdä perinteistä rekisteröitymistä.</li>
 <li>Menetelmä säilyttää kirjautumisen muissa <strong>OTP-</strong>moduuleissa saman istunnon aikana.</li>
 <li>Suositellaan erityisesti ulkopuolisille kurssin osallistujille ja suorittavan työn tekijöille.</li>
</ul>
