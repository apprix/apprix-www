<strong>Hotspot</strong> on yksi monikäyttöisimmistä elementeistä. Voit sijoittaa elementissä kuvan päälle, auki klikattavia, liikuteltavia kenttiä, ns. tärppikohtia, joita klikkaamalla suorittajalle avautuu tietoikkuna. Käyttötapoja ovat esimerkiksi:
<ul>
 <li>Elementin taustakuvana otsikot ja ikonit, joita klikkaamalla avataan ikonien symboloimia aiheita lisää.</li>
 <li>Kartta, josta suorittaja saa merkattuja alueita klikkaamalla niistä lisätietoa.</li>
 <li>Historiajana, josta vuosia klikkaamalla saa vuosien tapahtumista lisätietoa.</li>
 <li>Mosaiikki- eli yhdistelmäkuva eri aiheista, jos haluat paljon yksittäistä infoa samalle sivulle.</li>
 <li>Kuvankaappaus ohjelmasta tai esimeriksi laskusta, jossa autetaan merkittyjen kohtien avulla suorittajaa käyttämään/tulkitsemaan
asiaa.</li>
 <li>Kuva esimerkiksi hallin sisätiloista tai työmaa-alueesta, jossa suorittajaa ohjeistetaan etsimään ja klikkaamaan epäkohtia.</li>
</ul>
Jokaisesta klikattavasta kentästä avautuu ikkuna, johon voit valita sisällöksi kentän vasemmassa laidassa sijaitsevasta valikosta:
<ul>
 <li>tekstin</li>
 <li>kuvan</li>
 <li>videon</li>
 <li>totta/tarua tai oikein/väärin -tehtävän</li>
 <li>tai yhdistelmän edellä mainituista vaihtoehdoista.</li>
</ul>
Lisäksi voit valita, onko kenttä klikattava auki edetäkseen elementiltä (mandatory) vai ei.
<h4 class="fusion-responsive-typography-calculated" data-fontsize="20" data-lineheight="30px"><strong>Miten lisään kuvia Hotspot-elementissä lisättyihin</strong> kenttiin?</h4>
Kuvia lisätään hotspot-kenttiin samalla periaatteella kuin muihinkin elementteihin. Lue kuvien lisäämisestä lisää <strong><a href="#kuvien_lisaaminen_ elementteihin">täältä.</a></strong>
<h4 class="fusion-responsive-typography-calculated" data-fontsize="20" data-lineheight="30px"><strong>Miten lisään videoita</strong> Hotspot-elementissä<strong> lisättyihin kenttiin?</strong></h4>
Videot lisätään hotspot-kenttiin samalla periaatteella kuin Video-elementtiin. Lue aiheesta lisää <strong><a href="#videoelement">täältä.</a></strong>
<h4 class="fusion-responsive-typography-calculated" data-fontsize="20" data-lineheight="30px"><strong>Miten muutan hotspot-kentän</strong> sisällön<strong> True/False (Totta/Tarua) -tehtäväksi?</strong></h4>
True/False -ominaisuutta hyödynnetään samalla tavalla kuin <a href="#truefalse"><strong>True/False -elementtiä</strong></a>. Saat muutettua vastauksen väittämän oikeaksi tai vääräksi elementin <strong>ratassymbolista</strong> elementin edit-tilassa.
