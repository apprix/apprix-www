<h3>Info Click Titles Images</h3><p>In the <strong>Info Click Titles Images</strong> info element, you can divide a long subject area into a main heading, an introduction text shown below it, and clickable subheadings or chapters that the learner can press to get more information (sub-topics). This makes it easier for the learner to understand large amounts of text when it is divided into smaller sections. The learner first reads the topic introduction or general information on the left side of the element, then clicks the heading buttons below it and, after clicking, reads the additional information about the topic that opens on the right side of the element.</p><p>The clickable headings can be set as mandatory, in which case every subheading must be clicked open, or as optional, in which case these headings do not need to be clicked open to proceed past the task. Each subheading, when clicked active, has its own title field and body text field.</p><p>Below a sub-topic you can add either:</p><ul><li><p>an <strong>additional information button</strong> that can be used for organizing the sub-topic. The button can be named to suit the topic, such as "Did you know?" or "More information about topic X". When the learner clicks the button, an information box appears below the sub-topic.</p></li><li><p>an <strong>image field</strong>, allowing a freely chosen image to be placed below the sub-topic text.</p></li><li><p>Both functions can also be turned off, in which case the learner only sees the heading and the sub-topic text.</p></li></ul><h3>Element editing instructions</h3><ul><li><p>Write the main heading of the topic being covered and the topic introduction on the left side of the element. You can also leave the introduction field empty.</p></li><li><p>If a single-color background image has not been automatically set on the right side of the element: it is recommended to add a solid-color background on the right side so that the clickable headings and other text content stand out clearly (e.g. a white image if the texts are dark-colored). You can set the image by clicking the <strong>Change Image</strong> icon in the upper right corner of the left half of the element.</p></li><li><p>Add a background image to the right side of the element that supports the topic being covered.</p></li><li><p>Click <strong>Element Settings</strong> from the orange gear icon located at the top of the element's edit view. Add the required clickable topic headings in the <strong>Element Settings</strong> window. The topic heading buttons will appear on the left side of the element. Give the buttons titles according to the topics being covered.</p></li><li><p>Click the topmost heading button. Two small icons, <strong>Mandatory</strong> and <strong>Image</strong>, appear to the right of the button.</p></li><li><p>If you select:<ul><li><p>the <strong>Mandatory</strong> icon, opening the topic becomes mandatory (the learner must click it open before proceeding).</p></li><li><p>the <strong>Image</strong> icon, an image field appears below the sub-topic text field. Add an image to the image field that supports the topic covered above it.<ul><li><p>Note that when you select the <strong>Image</strong> option, the image field replaces the button that was in that position, which the learner could click to get additional information about the topic.</p></li></ul></p></li></ul></p></li><li><p>Add the chapter/sub-topic heading on the right side of the element, which can be the same as the clickable topic heading button. Write additional information about the topic in the text field below. You can also use this space for longer texts: if you do not use the image field or additional information button below, more space is freed up for the topic text field.</p></li><li><p>If you did not select the <strong>Image</strong> option, you can also use the button below to add additional information that the user does not need to read before proceeding.<ul><li><p>In that case, also name the button that the user clicks to open the text with additional information (for example "Did you know?" or "Read more").</p></li><li><p>Also add text to the text field that opens from the button.</p></li><li><p>Remove the text content from the heading button and the additional information field that opens from the button if you want to hide the button and the additional information field. If there is no text, the text box is not visible.</p></li></ul></p></li><li><p>Repeat these steps for each chapter (clickable sub-topic) you added in the <strong>Element Settings</strong> window.</p></li></ul><h3>Click &amp; Scroll</h3><p>The <strong>Click &amp; Scroll</strong> element improves upon the <strong>Click Title Images</strong> element. On the left side of the element you can add up to 20 clickable subheading buttons. Longer texts and images can be added to the right side of the element.</p><p><strong>Note! </strong></p><ul><li><p>Both sides of the element are scrollable.</p></li><li><p>If the element contains so much content that a scrollbar appears on the right side of a heading button, the learner must scroll down to ensure they have seen all the content. Only after that will the forward button appear for the heading button.</p></li></ul><p>To the right of each subheading button there are two icons:</p><ul><li><p><strong>Mandatory</strong>: If you want users to click and review the subheading content before proceeding, activate the icon on the left side. <strong>Note</strong>! If a subheading is marked as mandatory, it is marked as completed only when the user has scrolled to the end of the text that opens from it.</p></li><li><p><strong>Image</strong>: If you want to add an image at the bottom of the subheading, activate the icon on the right side.</p></li></ul><h3>Element editing instructions</h3><p>The editing instructions for this element are the same as for the Info Click Titles Images element.</p><h3>Element in preview mode</h3><p><img src="statamic://asset::files::docs/2021/08/Info_Click_Titles_Images_.gif"></p><h3>Element settings in edit mode</h3><p><img src="statamic://asset::files::docs/2021/08/Info_Click_Titles_Images_Edit_FI_.gif"></p><h3>Mandatory and Image icons</h3><p><img src="statamic://asset::files::docs/2024/10/Click-and-Scroll.png"></p>
