title: '5.1 Editing Text in Elements'
---
Click the element icon in the <b>Edit Module </b>view to open the element and edit its contents.

Edit all titles and text areas by clicking them. You can bold, italicize, underline, and change the color of the text from the text toolbar.

You can also create links by clicking the <b>Insert/edit link button</b>.

Copy the link inside the browser URL box if you want to create a hyperlink. Write an informative link title text (e.g.: Click here to read more about...) into Text to display field. 

<b>Tip!</b> Keep the texts as concise and simple as possible.
