title: '5.1 Editing Text in Elements'

<p>Go to the element you want to edit by clicking on it.</p><p>You can edit the headings and text on all element pages by clicking on them with the mouse. Text can be made bold, italic, underlined, and its colour can be changed. You can also create links via the "Insert/edit link" button in the text editor. In this case, enter the link in the "Url" field and, if desired, alternative display text in the "Text to display" field — for example, "Click here to learn more about the topic".</p><p><strong>Tip:</strong> Keep texts as concise as possible and remember to use paragraph spacing so that the text remains as clear as possible.</p><h3>How do you edit text in an element?</h3><p><img src="statamic://asset::files::docs/2021/08/Tekstin_editoiminen_elementissa.gif"></p>
