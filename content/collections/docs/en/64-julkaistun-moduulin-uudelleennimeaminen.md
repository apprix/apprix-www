title: '6.4 Renaming a Published Module'
---
In the Edit Module view, you can rename a module by clicking on its name and start editing. When the name is completed, click the <strong>Save</strong> button. A green pop-up window, "<strong>Saved!</strong>" will appear in the bottom right corner of the window.

Reload the page if you don't want to save a newly updated but unsaved module name. A message "<em>Do you want to reload the site</em>?" will appear, then click the Reload button.

<strong>Note!</strong>

- You can rename a module with the same name as an existing or "soft deleted" module. In these cases, you will be notified that a module with the same name already exists.

- The performance data will be kept in the same statistics table if you rename the module.
