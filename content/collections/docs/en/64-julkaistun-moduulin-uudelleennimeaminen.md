title: '6.4 Renaming a Published Module'
---
<p>In the Edit Module view, you can rename the module by clicking its name and starting to edit it. When the name is ready, click the <strong>Save</strong> button. A green popup "<strong>Saved!</strong>" will appear in the bottom-right corner of the window.</p><p>If you do not want to save the module name that has just been updated but not yet saved, reload the page. A message will appear in the window: "<em>Do you want to reload the site?</em>", then select the <strong>Reload</strong> button.</p><p><strong>Note!</strong></p><ul><li><p>You can rename a module with the same name as an existing or "soft-deleted" module. In these cases, you will receive a notification that a module with the same name already exists.</p></li><li><p>If you rename the module, the completion data is retained in the same statistics table.</p></li></ul><h3>Select the Reload button if you do not want to save the module name that has just been updated but not yet saved.</h3><p><img src="statamic://asset::files::docs/2024/05/reload-side.png"></p><h3>You will receive a notification if you name the module with the same name as an existing or deleted module.</h3><p><img src="statamic://asset::files::docs/2024/05/module-with-the-same-name.png"></p>
