title: '7.32 Landing Page'

<p>The purpose of the <strong>Landing Page</strong> element is to serve as a centralised collection page for courses. You can add multiple courses to the element and organise them, for example, by course topic or language. The element does not track or record courses that a user has completed — it simply acts as a kind of front page or link collection for courses.</p><p>A Landing Page module typically contains only Intro, Landing Page, and Finish elements, of which only the Landing Page element is visible in preview mode or to learners in the run view. This module is not designed to be completed; it serves only as a technical framework for the Landing Page element.</p><p>Whenever a learner selects a course from the Landing Page, a new browser tab opens for that course. Please note that to enable learner tracking, the modules added to the Landing Page must have a login feature added — the page itself does not collect learner data.</p><h3>Element editing instructions</h3><ol start="1"><li><p>Start by adding your organisation's logo, emblem, or other suitable image to the image field of the popup window displayed on top.</p></li><li><p>Welcome the learner in the text field below the image in the modal window shown at the top of the element, and provide, for example, general instructions for using the Landing Page and the courses.</p></li><li><p>Click the <strong>gear icon</strong> in the upper left corner of the page. The element settings will appear.</p></li><li><p>From the element settings, you can select the number of columns on the page (min. 1, max. 5). Columns divide the page, for example, by course category or topic area. Columns can also, for example, separate language versions of the same courses.</p></li><li><p>Once you have selected the number of columns, you can edit the column title and colour options in the popup window. Choose titles and colours for the columns. Then click <strong>Save</strong> to save your selections. Finally, select appropriate images for the columns.</p></li><li><p>Add buttons to columns by clicking the <strong>Add button</strong> button. Click the <strong>Header Here</strong> box that appears, and the button settings will open. Enter the course name (e.g. Safety Training). Leave the <strong>Disable</strong> checkbox unchecked if you want the button to be usable immediately after editing. You can also add a text that appears when the mouse cursor is moved over the course. This feature can be used by writing, for example, &quot;<em>Course not yet available</em>&quot; in the field if the course is not yet ready for use. Add the course's language/country code so the learner knows to select the right training if there are language versions of the courses, for example FI (Finnish), EN (English), EE (Estonian), etc.</p></li><li><p>Add the module's completion link to the <strong>Link</strong> field, so the button leads directly to the module. Save all changes by clicking <strong>Save</strong>. If needed, you can remove a button linking to a module by clicking <strong>Delete button</strong> in that button's settings. If you do not want to save changes made in the settings, click <strong>Cancel.</strong></p></li></ol><h3>Please also note the following usage recommendations:</h3><p><strong>Course module languages</strong></p><p>Courses can be available in several different languages: e.g. EN (English), FI (Finnish), EE (Estonian).</p><p>When a learner clicks a button to which a module is linked, a popup window opens. From the list shown in the window, the learner can select the language in which they want to complete the course. The course module opens in a new browser tab.</p><p><strong>Courses that are temporarily unavailable or still in progress</strong></p><p>Suppose you want to notify learners that a certain course module will be available soon. You can add the course completion link to the Landing Page by adding a button for it, selecting <strong>Disable button</strong>, and adding a mouseover text such as &quot;<em>Course available in month 202x</em>&quot;. The button is visible but cannot be clicked and its colour is faded. When the course is available, remove the <strong>Disable button</strong> option and the button will work normally.</p><h3>Example of a Landing Page element.</h3><p><img src="statamic://asset::files::docs/2025/05/LP-esimerkki-1.png"></p><h3>Landing Page - Button Settings</h3><p><img src="statamic://asset::files::docs/2025/05/LP-button-settings-1.png"></p><h3>How do you get started building a Landing Page element?</h3><p><img src="statamic://asset::files::docs/2025/05/LPStart-1.gif"></p><h3>How do you add a new button (course) to the Landing Page element?</h3><p><img src="statamic://asset::files::docs/2025/05/DLV-Start21-2.gif"></p><h3>How can you preview the content of a Landing Page element?</h3><p><img src="statamic://asset::files::docs/2025/05/LP-Preview-0105-1.gif"></p><h3>You can also use this element for storing links and documentation.</h3><p><img src="statamic://asset::files::docs/2025/05/DLP-for-links-and-documentation-1.png"></p>
