title: '7.32 Landing Page'
---
The <strong>Landing Page</strong> aims to centralise a collection of e-learning courses. This element allows you to organise several e-learning courses by topic and language. It doesn’t track which courses users have completed; it just lists the e-learning courses that are available.

By default, a landing page module contains only the Intro and Finish elements and the <strong>Landing page</strong> element between them. The trainee cannot continue to the finish element because s/he is supposed to attend one of the courses listed on the landing page instead. The trainees do not need to finish this module.

Every time the trainee selects a course, a new tab in the browser is opened. Please note that if statistical data needs to be collected from the courses, the e-learning course should contain the<b> Login </b>element or an authentication method (e.g., <strong>Email Registration</strong>) should be activated. In other words, statistical data cannot be collected from the Landing page module, as it is collected directly from the e-learning courses linked to the <b>Landing page</b>.
<h4 class="fusion-responsive-typography-calculated" data-fontsize="18" data-lineheight="27px">Editing Instructions</h4>
<ol>
 <li>Start by adding the Landing page image or a company logo.</li>
 <li><span>Write <strong>about how the <strong>landing</strong> page</strong> works in the introduction box and its purpose.</span></li>
 <li>Click the <b>gear</b> icon. The <b>Landing Page Settings </b>pop-up window is displayed.</li>
 <li>From the Landing page settings pop-up window, you can select the number of columns (max 5). The columns represent the training category types, e.g. Personnel, HSEQ, Contractors, etc</li>
 <li><span>Once you have selected the number of columns, the <strong>Columns header</strong> and<strong> Colour </strong>options will appear on the pop-up window. Edit the header title and choose the colours. Press the </span><b>Save </b>button. Now, the columns that you have selected are displayed in the Landing page window.</li>
 <li>You can add buttons to columns by clicking the <strong>Add button</strong>. After you have clicked the + <b>Add button</b>, the box <b>Header here</b> appears under the image. Click the <b>Header here</b> box. The Button settings pop-up window will open.</li>
 <li>Write the course header (course name), e.g. "Code of Conduct.</li>
 <li>Tick/untick the Disable button. This will disable the course button for the users.</li>
 <li>You can add the text displayed here once the user hovers over the box. For example, if the course is not yet available, in the Mouseover text box you could write the text "Not yet available" or "Available on (date here)"</li>
 <li>Add language abbreviation, e.g. EN (English), FI (Finnish), EE (Estonian), etc.</li>
 <li>Add the course link you get from Builder. If needed, you can also add any other training or document links.</li>
 <li>You can add another link using the same course button. For example, if you have multiple language versions of the course, you can add them all under the same course button (for example: "FI", "EN", "EE".</li>
 <li>Save all your changes by pressing the <b>Save</b> button.</li>
 <li>You can delete the whole button (course) by pressing the <b>Delete button</b>.</li>
 <li>If you don’t want to save your changes, you can close the pop-up by pressing <b>Cancel.</b></li>
</ol>
<h4 class="fusion-responsive-typography-calculated" data-fontsize="18" data-lineheight="27px">Please note the following:</h4>
<b>Course languages</b>

In the example, the course is available in EN (English) and FI (Finnish).

As displayed in the video clip on the right, a pop-up window opens when the user clicks the course button. The trainee can now choose his/her preferred language version of the course from the list. The course will open in a new tab.

<b>Disabled courses</b>

Suppose you want to inform trainees that a course will be available soon. You can add the course to the Landing Page, tick the <strong>Disable</strong> button, and write the needed information in the <strong>Mouseover text</strong> field, e.g., "<em>The course will be available by the end of March 202x.</em>"

When the trainee hovers with the mouse over the course, the Mouseover text, e.g. "<em>The course will be available by the end of March 202x</em>", is displayed, no pop-up window with the list of languages will open, and the course background colour is faded.
