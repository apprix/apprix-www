<p>Once a module has been published (<strong>Published</strong>), it starts collecting statistics. You can view the module's statistics by clicking the <strong>Stats</strong> button in the <strong>All Modules</strong> view or in the module's <strong>Edit Module</strong> editing view. The statistics show all completion data in real time.</p><p><strong>Note!</strong> Collecting user data requires that the module has a Login element, or another authentication method (e.g. Email Registration, Consumer Login, SSO/Single sign on, or Tax number check) enabled in the module settings (see the <strong>User Authentication</strong> tool – the gear symbol in the module editing view).</p><p><strong>Note!</strong> Data is collected separately for each module and submodule.</p><h3>Functions in the Stats window:</h3><ul><li><p>The <strong>Overview</strong> view shows a summary of the module's statistics.<ul><li><p><strong>Duration</strong> (the duration of the module/main module if the module has submodules) corresponds to the time the learner spent in the module.</p></li></ul></p></li><li><p>The <strong>Users</strong> tab shows all learners who have passed the module, i.e. gone through it to the end. You can also use the search field to filter learner names.</p></li><li><p>The <strong>Big Data</strong> tab allows you to view statistics for all learners who have participated in the course, as well as detailed, element-level statistics.<ul><li><p><strong>Show Finished</strong> button: shows only completions that have been successfully taken to the end of the module (<strong>Finished</strong> = 1)</p></li><li><p><strong>Show Unfinished</strong> button: shows both completions that have been successfully taken to the end and incomplete completions (<strong>Finished</strong> = 0)</p></li><li><p><strong>Show Archived</strong> button: If an element containing a task (e.g. True/False, CheckBox) has been removed from the module after the module was published, the data collected by these task elements is visible in the statistics when this option is selected.</p></li><li><p>In the <strong>Big Data</strong> tab you can also filter columns and download the data for analysis by clicking the <strong>Export</strong> button.<ul><li><p>You can also use the <strong>Search</strong> field to filter search results.</p></li></ul></p></li><li><p>The <strong>Progress</strong> column shows the learner's progress through the module (e.g. 3 = the 3rd element in the module).<ul><li><p>If the learner goes backwards in the module and then logs out, Builder saves the last "position" (Progress) in the statistics, i.e. the element they were on before logging out.</p></li><li><p>If they have progressed to the last element in the module (up to the <strong>Finish</strong> element), it is recorded as "<strong>Finished= 1</strong>". In this case, the "Finished= 1" entry does not change even if the user moves back from the last page and then logs out.</p></li></ul></p></li><li><p>The "<strong>Score</strong>"/"<strong>Max Score</strong>" column: the module's "learner's points" / "maximum points".<ul><li><p>Points are reset if the learner returns to the module later and restarts the module.</p></li></ul></p></li></ul></p></li><li><p>The <strong>Feedback</strong> tab: If the module contains a <strong>Feedback</strong> element for feedback, learners' feedback appears in the <strong>Feedback</strong> tab.<ul><li><p>Note! The <strong>Feedback</strong> tab also stores free-form feedback that learners have written in the <strong>Feedback</strong> text field to the right of the <strong>Feedback</strong> element.</p></li><li><p>If you want to view all feedback given in the Feedback element, open the <strong>Big Data</strong> tab.</p></li></ul></p></li></ul><p><strong>Tip!</strong> If your organisation also needs broader, customised completion reports, take a look at the <a href="/help/88-extended-reports"><strong>Extended Reports add-on feature.</strong></a></p><h3>You can view the module's statistics by clicking the Stats button.</h3><p><img src="statamic://asset::files::docs/2024/05/All-modules-stats.png"></p><p><img src="statamic://asset::files::docs/2024/05/Edit-module-stats.png"></p><h3>Where do you find the module's Big Data (statistics)?</h3><h3><img src="statamic://asset::files::docs/2021/02/Stats.png"><em>If the course module has been successfully completed to the end, the value in the Finished column is 1.
