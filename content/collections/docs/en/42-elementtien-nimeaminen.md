title: '4.2 Naming the Elements'
---
<p>Name the elements by clicking the text above the element. It is recommended to name the elements in a way that they describe the contents of the elements, for example, using the title used in the element content. This will make it easier to find the correct element later.</p>
When you're finished editing titles, make sure to save your changes by clicking <b>Save</b>.
