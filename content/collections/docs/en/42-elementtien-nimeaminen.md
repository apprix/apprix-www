<p>It is recommended to name the elements you use in a way that describes their content — for example, based on the main heading used in the element — so that editing the module is easier and recurring elements of the same type can be distinguished from one another without having to open each element.</p><p>Click <strong>Save</strong> after editing the name to ensure the changes are saved.</p><h3>How do you name an element?</h3><p><img src="statamic://asset::files::docs/2021/08/Elementin_nimeaminen.gif"></p>
