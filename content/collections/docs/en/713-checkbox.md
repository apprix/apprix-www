<p>In the <strong>Checkbox</strong> element, the learner is presented with several statements or questions at once and must select the correct answer from two options. The answer options for the question set can be defined as, for example:</p><ul><li><p>True/False</p></li><li><p>Yes/No</p></li><li><p>Correct/Incorrect</p></li></ul><p>Each statement has its own answer field, which allows a large amount of content to fit in the element. From the orange <strong>gear icon</strong> at the top of the edit view, you can define whether the statements or questions are true or false, change their order, and add or remove them.</p><p><strong>Tip:</strong> One common use case for the element is an end-of-module test. In newer versions of the element you can also add a larger number of questions than fit in the view, which are shuffled in random order so that only a portion of the questions are shown at any time.</p><p><strong>"Answer Mode" functionality</strong></p><p>If you want to activate the <strong>Answer Mode</strong> functionality, click the gear icon on the left edge of the window.</p><p><strong>Option 1: Answer Mode is turned off</strong></p><ul><li><p>Click the checkbox of each item. The answer text popup appears.</p></li><li><p>Edit its content.</p></li><li><p>Write additional information about the topic of the statement here. Note that the content is the same regardless of the answer given (e.g. True/False).</p></li></ul><p><strong>Option 2: Answer Mode is turned on</strong></p><ul><li><p>Enable Answer Mode by clicking the blue button in the settings bar.</p></li><li><p>Now click the checkbox of the item, and the answer text opens in a popup. CORRECT (Right) and INCORRECT (Wrong) options appear at the top of the popup. Provide justifications for each option.</p></li></ul><p><strong>NOTE!</strong> If the answer text in the popup is left empty, the answer text window will not appear when the learner has selected the answer checkbox.</p><h3>Element in preview mode</h3><p><img src="statamic://asset::files::docs/2021/08/Checkbox_FI_-1.gif"></p><h3>Element settings in edit view</h3><p><img src="statamic://asset::files::docs/2021/08/Checkbox_Edit_FI_-1.gif"></p><p><img src="statamic://asset::files::docs/2022/04/CheckBox.gif"></p>
