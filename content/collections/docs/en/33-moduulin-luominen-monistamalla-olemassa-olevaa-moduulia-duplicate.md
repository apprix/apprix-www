<p>Sometimes it may be more convenient to copy an existing module that resembles the desired end result rather than creating a completely new module. You can copy an existing module by clicking the <strong>Duplicate</strong> button to the right of the module in the Menu view. Builder will duplicate that module, after which you can start editing it however you like. Remember to name and save your copied module.</p><p>You can also copy an entire submodule structure using the Duplicate button of the main module in the Menu view.</p><h3>How is a module duplicated?</h3><p><img src="statamic://asset::files::docs/2024/07/Duplicate-new.gif"></p>
