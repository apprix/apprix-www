<p><strong>Survey Table</strong> is a survey element. The element allows you to present multiple survey-type questions that the learner must answer. A text box is shown at the start for the introduction, which explains the task and provides any additional information about the question or argument. After this, two answer options are available to choose from.</p><p>When the learner has selected their options, they click the forward button on the right side of the element. A new view opens. This view explains the differences between the options in the first and second columns, and provides, for example, additional information about the questions or arguments.</p><p>The survey result (the number of options selected from the first and second column) is shown, and the learner sees additional information below the survey results.</p><h3>Element editing instructions</h3><ul><li><p>Click the question mark icon in the upper right corner of the element. The task introduction text box opens. Write the survey title and scenario in the text field. If you want to show only the background image, you can remove the text. If there is no content, the text field will not be shown to the learner. Note that in edit mode you will still see the empty box with the text <em>&quot;- click to insert text -&quot;.</em></p></li><li><p>Click the <strong>Element settings</strong> option from the orange gear menu in the upper part of the element's edit view. Add, remove, reorder, and edit your questions in the <strong>Element settings</strong> window. New checkboxes will appear on the right side of the element.</p></li><li><p>Write the titles for the questions and options in the column fields.</p></li><li><p>Write the questions and their answer options (e.g. Option A, Option B). Write the answers as well if needed.</p></li><li><p>Click the forward button on the right side of the element. A new view opens. In this view the learner is shown the results of their choices.</p></li><li><p>Write additional information about the survey results in the upper text box. If you want to show only the background image, you can remove the text. If there is no content, the text field will not be shown to the learner. Note that in edit mode you will still see the text <em>&quot;- click to insert text -&quot;</em> in the text field, to make any further editing easier.</p></li><li><p>The middle box lists the survey results.</p></li><li><p>Write additional information in the lower text box or leave it empty.</p></li><li><p>Add a background image by clicking the <strong>Change image</strong> icon in the upper right corner of the element's image field.</p></li></ul><p></p><h3>Element in published mode</h3><p><img src="statamic://asset::files::docs/2021/08/Survey_Table_FI-1.gif"></p><h3>Element settings in edit view</h3><p><img src="statamic://asset::files::docs/2021/08/Survey_Table_Edit_FI_Small-1.gif"></p><p><img src="statamic://asset::files::docs/2021/06/s1.png"></p><p><img src="statamic://asset::files::docs/2021/06/s2.png"></p><p><img src="statamic://asset::files::docs/2021/06/s3.png"></p><p><img src="statamic://asset::files::docs/2021/06/s4.png"></p><p><img src="statamic://asset::files::docs/2021/06/s5.png"></p>
