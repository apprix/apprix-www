title: '7.37 Survey Table'
---
<strong>Survey Table</strong> is a survey element. In this element, you may ask several questions which the trainee must answer by choosing from two available options. At the beginning, an info message is displayed explaining the task instructions and giving additional information about the topic. 

Once the options have been chosen, the trainee then clicks the move forward arrow button on the right side of the element. A new view is opened. In this new view, the differences between the options in the first and second columns are explained. Also additional information about the topic is given.

The survey result (number of options chosen from the first and second column) is revealed and additional information is displayed below the results.
<h4>Editing Instructions</h4>
<ul>
 <li>Add a background image by clicking the <b>Change image</b> icon from the top right corner of the background image. </li>
 <li>Click the <b>Question mark icon</b> on the top right of the element view. A text box is opened. Write the title and the scenario for the survey in the text box. 
<ul>
 <li>In case you only want to show the background image, you can remove the text. If there is no content, the text box will not be displayed to the trainee. Please note that for editing purposes, you will still see an empty text box with the text <i>"- click to insert text -".</i> </li>
</ul>
</li>
 <li>Click the <b>Element settings</b> option from the gear icon at the top of <strong>Edit Element Contents</strong> view. 
<ul>
 <li>In the window, add and edit your survey questions. New option boxes will appear on the right side of the element.</li>
</ul>
</li>
 <li>Write the titles of the questions and options in the <b>Column</b> fields. </li>
 <li>Write the questions and their options texts (e.g. Option A, Option B). Write the answers too, if  needed.</li>
 <li>Click the forward button on the right side of the survey table. A new element view is opened. 
<ul>
 <li>In the new view, the results of their selections are shown to the trainee.</li>
</ul>
</li>
 <li>Write any additional information about the survey results in the upper text box. 
<ul>
 <li>In case you only want to show the background image, you can remove the text. If there is no content, the text box will not be displayed to the trainee. Please note that for editing purposes, you will still see an empty text box with the text <i>"- click to insert text -".</i> </li>
</ul>
</li>
 <li>The results of the survey will be listed in the text box in the middle.</li>
 <li>Write any additional information in the lower text box.
<ul>
 <li>If you leave the box empty, it will not be visible to the trainees.</li>
</ul>
</li>
</ul>
