title: '7.24.5 What if the browser does not support the mp4 video format?'
---
<p>The MP4 video format is supported by almost all modern browsers. You can also upload a webm or ogg video file alongside the MP4 file to support other browsers, such as Opera Mini.</p>

<p>If you want to upload multiple video files in different formats at once, click the "<strong>Upload new video files…</strong>" button, which opens a file picker window (File Explorer on Windows, Finder on macOS). Select multiple files, e.g. MP4 and webm files, by holding down the CTRL key on Windows or the CMD key on macOS and selecting the required files. The maximum size per file uploaded is 100 MB.</p>

<p><strong>Note!</strong> Always remember to add a video in MP4 format. You cannot add webm or ogg video files without an MP4 file.</p>
