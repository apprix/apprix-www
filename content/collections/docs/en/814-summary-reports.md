The <strong>Summary Reports</strong> view aims to combine modules' performances in one place so facilitators can monitor them.

The access rights are granted in the <strong>Administrate</strong>  --&gt; <strong>Users</strong> tool.

<strong>Note</strong>! The facilitator can also grant external users access rights to the Report tool.

The external user facilitator can be set in the <strong>Administrate → Users</strong> tool to <strong>Module Access</strong> = <strong>None</strong> and <strong>Summary Report</strong> = <strong>On</strong>, in which case they only have access to the reporting.

When this option is enabled, a new <strong>Reports</strong> button will appear in the Builder <strong>All Modules</strong> tab.

Once the <strong>Reports</strong> button is clicked, the reporting window opens. The <strong>Filters</strong> button in the top left corner of the window allows the facilitator to select several filters.

The filters are customer-specific. They can be, for example, <strong>Emails</strong> (the email address of the person completing the course), <strong>Modules</strong> (only data from published modules are available in the reports), <strong>Date Range</strong> (the date range of completions), or <strong>Finished</strong> (if selected, it shows only those completions that have completed the training; if not selected, it shows all completions, including incomplete ones).

Once the facilitator has set the filters and obtained the desired report, they can download it in <strong>CSV</strong> or <strong>Excel</strong> format.

<strong>Note!</strong>

A more advanced version of Summary Reports is the E<strong>xtended Reports</strong> feature.
