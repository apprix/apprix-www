<p>The <strong>Edit Points</strong> feature is found in the dropdown menu that opens from the <strong>gear symbol</strong> in the module editing view by clicking <strong>Edit Points</strong>. Score calculation happens automatically in percentage units, and points do not normally need to be changed when creating a new module. The maximum result is 100% and it is automatically divided among the task elements added to the module.</p><p>If you wish, you can change the relative importance of tasks. By default, each task is worth 1 point, i.e. equally important in terms of scoring value. You can make a task more important in terms of scoring by increasing its point value. You can also remove a task's point value by deleting the task's points and replacing them with 0 points.</p><p><strong>Note!</strong> The <strong>Edit Points</strong> view also allows you to edit the module's percentage pass threshold.</p><p>If the module has a final test towards the end along with a <a href="/en/help/730-passfail-and-passfail-always"><strong>Pass/Fail element</strong></a> that controls passing, and only the test scores should be taken into account in the score calculation, remove the points from all elements except those belonging to the test and set the percentage pass threshold to the desired level.</p><h3>How do you edit element points and the percentage of points required to pass the module's final test?</h3><p><img src="statamic://asset::files::docs/2024/08/Edit-Points.gif"></p>
