<p>When you log in, you see the <strong>Menu view</strong>, which lists all created courses, i.e. modules. In this view you can:</p><ul><li><p>create a new module from the <strong><a href="/help/31-moduulin-luominen-create">Create sidebar</a></strong></p></li><li><p>copy an existing module by clicking the <a href="/help/33-moduulin-luominen-monistamalla-olemassa-olevaa-moduulia-duplicate"><strong>Duplicate</strong></a><strong> button</strong> on the right side of the module you want to copy</p></li><li><p><p>search for modules using the <strong>Search</strong> function or <a href="/help/68-moduulien-ryhmittaminen-ja-moduulien-hakemisen-helpottaminen-tag-moduulitunni"><strong>Tags</strong></a><strong> (module labels)</strong></p><ul><li><p>Tags can be used to label modules, making them easier to find by using the tag as a search criterion.</p></li><li><p>A tag can be, for example, a language, department, location, or training topic.</p></li></ul></p></li><li><p><p>browse, edit, and view existing modules <strong>to which you have either view or edit permissions</strong></p><ul><li><p>For published modules you will see the <a href="/help/62-moduulin-julkaiseminen-ja-suorituksia-keraava-moduulilinkki-publish-ja-link"><strong>Published</strong></a><strong> button</strong></p></li><li><p>The <a href="/help/61-moduulin-esikatseleminen-preview"><strong>Preview</strong></a><strong> button</strong> opens a preview of the module</p></li><li><p>If you have edit and/or statistics viewing permissions for a module, you may also see the <a href="/help/63-tilastointi-stats"><strong>Stats</strong></a><strong> button</strong>, from which you can view the module's completion statistics.</p></li></ul></p></li></ul><p>If you are one of your organisation's administrators, you may also see the <a href="/help/82-user-authentication-kayttajatodennus"><strong>Administrate sidebar menu</strong></a> on the left side of the view, from which you can edit user management settings.</p><h2>How to use the 'Quick filters' tool to search for modules?</h2><p>The 'Quick filters' tool allows you to sort and filter modules and online courses in Apprix Builder. This makes it easier to find the right module quickly and efficiently.</p><p><strong>What can you do?</strong></p><p>With the 'Quick filters' tool you can:</p><ul><li><p><strong>sort modules</strong> alphabetically or by date</p></li><li><p><strong>filter your own modules</strong>, i.e. those you have created yourself</p></li><li><p><strong>filter editable modules</strong> to which you have edit permissions</p></li><li><p><strong>filter commented modules</strong> that have unresolved comments</p></li><li><p><strong>filter by publication status</strong>: all, published, unpublished</p></li></ul><p><strong>Where can you find the feature?</strong></p><p>The <strong>'Quick filters'</strong> tool is located in the <strong>top right corner</strong> of Builder's <em>Menu</em> &gt; <em>All Modules</em> window. Click the icon to open the filter menu. The selected filters are displayed in the filter text field in the top left corner of the <em>All Modules</em> window.</p><p><strong>Tip:</strong> If you are looking for a specific module, try combining multiple filters — for example, your own unpublished modules that have comments.</p><h3>Menu view with course modules filtered by name.</h3><p><img src="statamic://asset::files::docs/2025/11/All-Modules-Menu-with-the-name-filter.png"></p><h3>Menu view with course modules filtered using the Quick Filters tool</h3><p><img src="statamic://asset::files::docs/2025/11/All-Modules-Menu.png"></p><h3>Quick Filters tool</h3><p><img src="statamic://asset::files::docs/2025/11/sort-and-filter-modules-tool.png"></p>
