<p>By following these instructions you can add links and files to the module that are available to the learner throughout the entire module.</p><h3>What is the hamburger menu?</h3><p>The <strong>hamburger menu</strong> button is located in the top-right corner of elements. Clicking the menu opens a list of files and links that are available to learners completing the training.</p><h3>Editing instructions</h3><p>When you click the hamburger menu, the <strong>Link menu settings</strong> popup window opens. In this window you can add <strong>links</strong> (https://www.google.com/) and <strong>files</strong> (pdf, doc/docx, xls/xlsx, ppt/pptx, txt/csv, zip) to the hamburger menu.</p><p><strong>Note!</strong></p><p>Links and files that are</p><ul><li><p>added or removed via the hamburger menu of any element are automatically updated in every element of the module.</p></li><li><p>added to the main module are not automatically updated in submodules, or from a submodule to the main module and other submodules, as the hamburger menu is module-specific.</p></li></ul><h3>+ ADD LINK button</h3><ul><li><p>Click the <strong>+ ADD LINK</strong> button when you want to add links to the hamburger menu. The <strong>Browse Media Library</strong> popup window opens. In this window you can see a list of previously added links, if any exist.</p></li><li><p>If the link you are looking for is not yet in the window's list, click the orange <strong>Add new link...</strong> button. A new popup window opens asking you to provide a name for the link. This name will appear as the link's name in the window's list. Enter a name for the link (e.g. google.com) and click <strong>OK</strong>.</p></li><li><p>A new popup window opens again, asking whether you want to keep the link in the module's link library, so that the link appears in the list of added links. Enter a name for the link (e.g. google.com) and click <strong>OK</strong>. The popup window and the <strong>Browse Media Library</strong> window close.</p></li><li><p>Click the <strong>+ ADD LINK</strong> button again in the <strong>Link Menu Settings</strong> menu. In the <strong>Browse Media Library</strong> popup window, select the desired link and click the <strong>Insert Selected</strong> button in the bottom-right corner of the window. The link will now appear in the <strong>Link Menu Settings</strong> window.</p></li><li><p>Select <strong>Done</strong> if you do not want to add more links. The link is now available from the <strong>hamburger menu</strong>.</p></li></ul><h3>+ ADD FILE button</h3><ul><li><p>Click the <strong>+ ADD FILE</strong> button when you need to add a file to the hamburger menu. The <strong>Browse Media Library</strong> popup window opens. In this window you can see a list of previously added files, if any exist.<ul><li><p>If the file you are looking for is not yet in the window's list, click the orange <strong>Add new file...</strong> button.</p></li></ul></p></li><li><p>Note! Check the <strong>Show uploaded file in library</strong> checkbox if you want the file to appear in the file listing in the window, i.e. in the file library.</p></li><li><p>The <strong>File Explorer</strong> window opens. Select the file you want to add to the library. A new popup window opens asking you to provide a name for the file so it can be found in the library. Enter a name for the file and click <strong>OK</strong>. The popup window and the <strong>Browse Media Library</strong> window close.</p></li><li><p>Click the <strong>+ ADD FILE</strong> button again in the <strong>Link Menu Settings</strong> window. Select the file in the <strong>Browse Media Library</strong> window and click the <strong>Insert Selected</strong> button in the bottom-right corner of the window. The file will now appear in the <strong>Link Menu Settings</strong> window.</p></li><li><p>Select <strong>Done</strong> if you do not want to add any more files. The file has now been added and is visible in the <strong>hamburger menu</strong>.</p></li></ul><p>The files and links added to the <strong>Link Menu Settings</strong> menu can now be viewed from all elements in the module via the <strong>hamburger menu</strong>. If you want to remove a file or link, click the <strong>trash icon</strong> to the right of the link/file name.</p><p>If you want to remove a link or file from the <strong>Media Library</strong> window's list, click the <strong>REMOVE SELECTED FROM LIBRARY</strong> button in the bottom-left corner of the window. Please note that the link/file is then only removed from the list visible in the <strong>Media Library</strong> window, but not from the module if the link or file has been added from the element's <strong>Link Menu Settings</strong> popup window.</p><h3>How do you add links to the hamburger menu?</h3><p><img src="statamic://asset::files::docs/2024/07/Hamburger-link.gif"></p><h3>How do you add files to the hamburger menu?</h3><p><img src="statamic://asset::files::docs/2024/07/Hamburger-file.gif"></p>
