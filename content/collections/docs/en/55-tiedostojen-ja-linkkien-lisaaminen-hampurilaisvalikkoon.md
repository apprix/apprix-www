Follow these instructions to add files and links that are to be available to trainees throughout the module.
<h4>What Is the Hamburger Button?</h4>
The <strong>hamburger button </strong>(aka hamburger menu) is placed at the top right corner of an element. Clicking on the button opens a list of files and links, ready to be used by trainees during the training.
<h4><b>Editing Instructions</b></h4>
Clicking on the <b>hamburger button</b> opens the <b>Link Menu Settings</b> pop-up window. In this window, <b>links </b>(e.g. <i>HTTPS://www.google.com/)</i> and <b>files</b> can be added (pdf, doc/docx, xls/xlsx, ppt/pptx, txt/csv, zip) into the hamburger.

<b>Note!</b>

Links and files
<ul>
 <li>placed in or removed from the <b>hamburger button</b> of any element are updated accordingly in all elements in the module automatically.</li>
 <li>added in the main module are not updated in the submodule and vice versa, since these are module-specific.</li>
</ul>
<h4><b>+ ADD LINK Button</b></h4>
<ul>
 <li>Click the <b>+</b> <b>ADD LINK </b>button if you need to add a link(s) into the hamburger. The <b>Browse Media Library</b> pop-up window will be opened. In this library, you can find a list of available links added previously. </li>
 <li>If the link that you need is not included in the library yet, click the orange <b>Add new link</b>… button. A new pop-up window is opened; it requests to provide a name for the link to be displayed in the media library. Enter the name of the link, e.g. <i>google.com </i>and press <b>OK</b>. </li>
 <li>A new pop-up window is opened, it requests to give the link to be stored in the media library. Type the link name, e.g. <a href="https://www.google.com/">https://www.google.com/</a> and press <b>OK</b>. The pop-up window and the <b>Browse Media Library</b> are closed. </li>
 <li>In the <b>Link Menu Settings</b>, click the <b>+</b> <b>ADD LINK </b>button again. In the <b>Browse Media Library</b> pop-up window, select the needed link and click <b>Insert Selected</b> button at the bottom right side of the window. The link now will appear in the <b>Link Menu Settings</b> pop-up window. </li>
 <li>Select <b>Done</b> if you do not need to add more links. The link is now available in the <b>hamburger button</b>.</li>
</ul>
<h4><b>+ ADD FILE Button</b></h4>
<ul>
 <li>Click the<b> +</b> <b>ADD FILE</b> button if you need to add a file into the hamburger button. The <b>Browse Media Library</b> pop-up window will be opened. In this library, you can find a list of available files, added previously. 
<ul>
 <li>If the file that you need is not included in the library yet, click the orange <b>Add new file</b>…button. </li>
</ul>
</li>
 <li><b>Note!</b> Tick the box <b>Show uploaded file in library</b> if you wish that the file is included in the <b>File Library</b>.</li>
 <li>The <b>File Explorer</b> window is opened. Select the file to be included in the media library. </li>
 <li>A new pop-up window is opened; it requests to provide a name for the file to be displayed in the media library. Enter the file name and press <b>OK</b>. The pop-up window and the <b>Browse Media Library</b> are closed. </li>
 <li>In the <b>Link Menu Settings,</b> click the <b>+</b> <b>ADD FILE </b>button again. In the <b>Browse Media Library</b> pop-up window, select the needed file and click <b>Insert Selected</b> button at the bottom right side of the window. The file will appear in the <b>Link Menu Settings</b> pop-up window. </li>
 <li>Select <b>Done</b> if you do not need to add more files. The file is now available in the <strong>h</strong><b>amburger button</b>.</li>
</ul>
Files and links included in the <b>Link Menu Settings </b>are now available in all elements of the module. If you need to remove a file or a link, click the <strong>T</strong><b>rash Can</b> icon on the right side of the link/file name.

If needed, you can also remove a file/ link from the <b>Media Library</b>, by selecting the file/link and clicking <b>REMOVE SELECTED FROM LIBRARY</b> button at the bottom left side of the pop-up window. Note that the file/link will be removed only from the <b>Media Library</b>, but not from the module(s), if the file/link is stored in the element <b>Link Menu Settings </b>pop-up window.
