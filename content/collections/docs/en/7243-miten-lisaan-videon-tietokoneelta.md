title: '7.24.3 How do I add a video from my computer?'
---
<ol>
 <li>Open the Video element.</li>
 <li>Click the "Add your video here" window. The <strong>Change Video/Audio</strong> popup appears.</li>
 <li>Select the <strong>Files</strong> tab and click <strong>Browse/Upload Files</strong></li>
 <li>The <strong>Browse Media Library</strong> popup appears. If you want to save the video file to the media library, first check the <strong>Show uploaded file in library</strong> box<strong>.</strong></li>
 <li>Click <strong>Upload new video file</strong> and give the video a name. Upload the file.</li>
</ol>

<p>The MP4 file format is supported by almost all browsers and their latest versions. To get more information about MP4 video files, read more about the topic: <a href="https://caniuse.com/#feat=mpeg4">https://caniuse.com/#feat=mpeg4</a></p>

<p>The maximum size for videos uploaded to Builder is 100MB.</p>
