<p class="p1">In the <b>True/False</b> element, you ask a question or provide a statement which can be set as correct or incorrect from the <b>gear icon</b> in the <b>Edit Module Contents</b> view. </p>
Common uses for the element:
<ul>
 <li>True or False statements</li>
 <li>Yes or No questions</li>
 <li>Do / Don’t statements</li>
</ul>
<b>Tip! </b>These tasks are among the easiest to create because you don’t have to create incorrect options. When you have longer info paragraphs, see if you can create a statement from the beginning and use the rest of the paragraph as an answer text.
