<p><strong>Phrases</strong> is a task element in which the learner must drag word tiles in the correct order from the starting box at the top to their positions in the sentence in the ending box below. A word tile can contain one or more words of text, and some of the tiles may also be incorrect — meaning they do not belong in the sentence.</p><p>After the learner has moved the required word tiles to the ending box, they click the forward button on the right, and Builder checks whether the sentence the learner has constructed is correct. The starting box is replaced by an answer box (red if the answer is incorrect, green if correct), giving the learner feedback. The correct sentence appears in the ending box.</p><p>If the answer is 100% correct, an <strong>OK</strong> icon appears. If the answer is incorrect, a <strong>Not OK</strong> icon appears.</p><p><strong>Here are some ideas for when to use this element:</strong></p><ul><li><p>when building a company's mission or vision</p></li><li><p>in tasks that require precise wording, e.g. in radio communications.</p></li></ul><h3>Element editing instructions</h3><p>Add a background image to the element that supports the topic being covered.</p><p>Click the <strong>Element Settings</strong> option from the orange gear icon in the upper part of the element's edit view.</p><p>Add the required draggable word tiles from the <strong>Element Settings</strong> window. The word tiles will appear on the right side of the element. Name the tiles in logical order, e.g. "<em>A Code of Conduct</em>", "<em>is a set of values</em>", "<em>and principles</em>", "<em>that guide</em>", "<em>an organisation.</em>"</p><p>If needed, add the incorrect word tiles below the last correct tile, for example in this case "<em>A Code of behaviour</em>", "<em>is a set of laws</em>", and "<em>that run</em>". In this example, the learner should move the tiles that form the following sentence: <em>&quot;A Code of Conduct is a set of values and principles that guide an organisation.&quot;</em></p><p>Enable the <strong>Shuffle</strong> function. If the function is not activated, the word tiles will appear in the same order in the starting box.</p><p>The green answer box will appear when you click the forward button. Write the answer text here, which gives additional information about the correct sentence. The same text also appears on the red side.</p><p>In the upper left text field, provide information about the sentence and instructions for the learner, for example:</p><p><em>"Drag the word tiles from the upper right in the correct order into the box below. Click the forward button when you are ready to see the result."</em></p><p>You can adjust the difficulty of the task by clicking the tool icon in the upper left corner. The <strong>Phrases element settings</strong> will then open.</p><p>If the option "<em>Allow only the right number of words to be dragged</em>" is activated in the Phrases element settings, the correct number of tiles to be moved is shown in the lower right corner of the window. If this option is not selected, the forward button is shown after the first word tile has been moved to the lower box.</p><p><strong>Scoring logic</strong></p><p>The element calculates how correctly the task was completed as a percentage (0–1, 1 = 100%). The percentage is recorded in the statistics. The percentage is multiplied by the maximum points assigned to the element to give the points earned from the element.</p><p>The percentage is calculated by examining three things:</p><ul><li><p>how many correct words the learner selected</p></li><li><p>how many incorrect words the learner selected</p></li><li><p>whether the entire phrase is in the correct order</p></li></ul><p><strong>Normal option scoring logic:</strong></p><ul><li><p>If the entire phrase is correct, the learner receives 100% for the task</p></li><li><p>If there are any errors in the answer (wrong order, too few or too many words selected), the user is given 50% of the maximum points.</p></li><li><p>From this 50%, percentage points are deducted for each incorrectly dragged word.</p></li><li><p>From this 50%, each incorrectly selected word reduces the percentage by which the element's maximum points are multiplied.</p></li><li><p>The proportional percentage value of one incorrect word is the same as that of one correct word, meaning one incorrect word deducts the same percentage as one correct word adds.</p></li></ul><p></p><h3>Element in published mode</h3><p><img src="statamic://asset::files::docs/2024/12/Phrases-element-demo.gif"></p><h3>Element in edit mode</h3><p><img src="statamic://asset::files::docs/2024/12/Phrases-element-settings-image.png"></p><h3>Element settings in edit mode</h3><p><img src="statamic://asset::files::docs/2024/12/Phrase-asetukset.png"></p>
