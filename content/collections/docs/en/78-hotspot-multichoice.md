<p><strong>Hotspot Multichoice</strong> is a multiple-choice task element. The learner first reads the task instructions from the <strong>Introduction</strong> box, then the question and additional information from the info boxes.</p><p>The learner must then answer by selecting the options (hotspots) they believe are the correct answers to the question.</p><p>Once the options have been selected, the learner clicks the forward arrow on the right side of the page. Hotspots that are correct answers will show a checkmark. If additional information about an answer has been added in edit mode, a short message appears near the hotspot containing an explanation of why that hotspot is correct or incorrect.</p><p>A summary box can be added at the end of the task to present a summary of the task result. If the box is left empty in edit mode, the window will not appear and learners will only see the Forward button to exit the task.</p><h3>Element editing instructions</h3><ul><li><p>Write the introduction and task summary in the <strong>Introduction</strong> and <strong>Summary</strong> boxes if you want to show an introduction at the start of this task and a summary at the end. You can also leave the boxes empty — in that case the boxes will not be visible to the learner.</p></li><li><p>Place the movable fields for the multiple-choice options at the desired location on the background image.</p></li><li><p>Write the task question, scenario, or situation in the text boxes. If you want to show only the background image, you can remove the text. If there is no content, the text boxes will not be shown to the learner. Note that in edit mode you will still see the empty boxes with the text <em>"- click to insert text –".</em><ul><li><p>If you want to hide the boxes in edit mode, click the round <strong>Hide Infos</strong> icon in the upper right corner.</p></li></ul></p></li><li><p>Add a background image by clicking the <strong>Change image</strong> icon in the upper right corner of the background image.</p></li><li><p>Click the <strong>Element settings</strong> option from the orange gear menu in the upper part of the element's edit view. A new window opens. Add the required hotspot fields and set them as correct or incorrect options.<ul><li><p>Drag the hotspot field into position by grabbing it with the mouse and placing it.</p></li><li><p>You can adjust the size and shape of the hotspot from its edges.</p></li><li><p>Change the shape using the handle on the right side of the hotspot field.</p></li></ul></p></li><li><p>You can now add answer text to each hotspot field's message box individually by clicking the hotspots. Write a detailed explanation in each hotspot of why the option is correct or incorrect. If you leave this text box empty, it will not be shown to learners when they check their answers.</p></li><li><p>Click the orange tool icon in the upper left corner of the window. A toolbar for adjusting the position and colour of the text boxes will appear.</p></li><li><p>Below the toolbar on the left side there is the <strong>Hide hotspots</strong> function.</p></li><li><p>In the upper right corner of the element there is a round <strong>Intro/Summary</strong> button.</p></li><li><p>To make editing the element easier, you can hide the info boxes in edit mode by clicking the <strong>Hide/Show Infos</strong> button in the upper right corner.</p></li></ul><h3>Element in published mode</h3><p><img src="statamic://asset::files::docs/2024/08/Hotspot-multichoice-video-optimized.gif"></p><h3>Element settings in edit mode.</h3><p><img src="statamic://asset::files::docs/2024/08/Hotspot-multichoice-video-settings-opt2.gif"></p><h3>You can add answer text to each hotspot field's message box individually by clicking</h3><p><img src="statamic://asset::files::docs/2024/08/Hotspot-multichoice-video-items-opt-3-1.gif"></p>
