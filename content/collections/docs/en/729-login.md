<strong>Login</strong> elements are always customized for the individual needs of each client. Some Login elements may include integrations, but most often they only gather additional information about the trainees for statistics purposes. User authentication and course access are usually defined separately in the module's settings (e.g. <a href="#sso"><strong>SSO</strong></a>, IP guard, Email registration).

Please note that the trainees need to be identified so that a Builder module is able to collect identifiable data about them. Usually, this is achieved by a module-specific access setting such as <strong><a href="#sso">SSO</a></strong>, but if your Login element has been customized to ask for a unique identifier (e.g. email, tax ID), you may be able to use it for the same purpose. You may find additional information about your customized Login element by going to the element's <strong>Edit Element Contents view</strong>.

<strong>Tip</strong>: The <strong>Form</strong> element offers an improved version of the Login element.
You can find more details at the link below:
<a href="https://faq.apprix.fi/builder-help-en/#form-element">https://faq.apprix.fi/builder-help-en/#form-element</a>
