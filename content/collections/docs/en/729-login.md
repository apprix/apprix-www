<p>The <strong>Login</strong> element is customized individually for each customer. Sometimes Login elements include technical integrations with other systems, but most commonly the element collects additional information about learners for statistical purposes. User authentication and login to a course module is generally configured separately in the module settings (e.g. SSO, IP Guard, email registration).</p><p>Please note that the learner must be identified in order to collect individually identifiable statistics about them. Most commonly this is achieved through a module-specific setting that can be enabled, such as <a href="/help/82-user-authentication-kayttajatodennus"><strong>SSO</strong></a>. If the Login element has been customized to require a unique identifier (e.g. email address, tax identification number), it may also be used for the purpose of identifying the learner. You can get more information about your organization's customized Login element by examining it in its edit view.</p><p><strong>Tip:</strong> The <strong>Form</strong> element is an improved version of the Login element.
