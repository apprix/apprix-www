---
The <b>Submenu Buttons element</b> is a contents menu page located in the main module from where the trainee can access submodules. The element is usually added right after module instructions (usually Intro and Login elements). When a submodule has been finished, the trainee returns to this element. When all submodules have been finished, the trainee can continue the main module and proceed to finish the training.

Read more about the benefits of using submodules and when it is recommended to create a submodule structure <b><a href="#creatingsubmodulestructures">here</a></b>.
