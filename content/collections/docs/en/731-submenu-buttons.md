<p>The <strong>Submenu Buttons element</strong> is a table of contents menu for a long module structure, from which the learner can navigate from the main module to submodules that focus on specific topic areas. The submenu is added to the main module after the Intro and any Login element, but before any final test.</p><p>When a submodule has been completed, the learner is directed back to the menu. Read more about the benefits of the submodule structure and when it is recommended to use it <a href="/en/help/34-submoduulirakenteen-luominen-paamoduuli-submoduulit">here.</a></p><h3>Submenu Buttons in edit mode</h3><p><img src="statamic://asset::files::docs/2020/04/Submodule_buttons.gif"></p>
