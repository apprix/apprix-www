title: '3.2 Naming a Module'

<p>Start by naming the module in the top left corner by clicking on the text. It is recommended to name the module in a way that describes its purpose, for example "Safety Training" if the training covers safety topics. If language versions of the training are planned, you can also add a country code such as FI, EN, etc. after the training name, which makes it easier to quickly find the module you are looking for. Finally, press <strong>Save</strong> to save the name change.</p><h3>How is a new module named?</h3><p><img src="statamic://asset::files::docs/2021/08/Moduulin_nimeaminen.gif"></p>
