<p><strong>Login with PIN code</strong></p>

<p>This login method removes the need to remember passwords. The course participant simply enters their basic information (first name, last name, and email address) in the <strong>Welcome window</strong> each time they start the course.</p>

<p><strong>Activating the "Email One-Time PIN (OTP)" authentication method</strong></p>

<ol>
 <li>Open the module in <strong>Edit mode (Edit module)</strong>.</li>
 <li>From the <strong>settings (gear) icon</strong> at the top of the module, select <strong>User Authentication…</strong>. The <em>User Authentication &amp; Access</em> dropdown opens.</li>
 <li>Select <strong>Email One-Time PIN (OTP)</strong>. The selection is now visible in the gear dropdown.</li>
</ol>

<p><strong>How it works</strong></p>

<ul>
 <li>The course participant enters their basic information (email address, first name, last name) in the <strong>Welcome window</strong> and then clicks the Send Pin Code button. The PIN code is generated automatically by the authentication platform.</li>
 <li>The PIN code is sent to the participant by <strong>email</strong>.</li>
 <li>The participant pastes the PIN code into the <strong>authentication popup</strong>.</li>
 <li>The course is ready to begin.</li>
</ul>

<h3>Note! The user's email address, first name, and last name must be unique.</h3>

<p><img src="/files/docs/2025/12/pin-code-message-1.png"></p>

<p><strong>Important notes</strong></p>

<ul>
 <li>A new, unique <strong>PIN code</strong> is sent with each login.</li>
 <li>The code must be entered within <strong>10 minutes</strong> of receipt.</li>
 <li>The user does not need to complete a traditional registration.</li>
 <li>The method maintains the login in other <strong>OTP</strong> modules during the same session.</li>
 <li>Particularly recommended for external course participants and frontline workers.</li>
</ul>
