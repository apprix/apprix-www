<p>In the <strong>Upload</strong> element, the learner can upload their own personal photo to the Apprix server for an ID card while completing the training.</p><p>Clicking the "<strong>1 Select image</strong>" icon opens the computer's file browser from which a photo can be uploaded. If the learner is using a mobile phone or tablet, they have the following options: "Take photo or video", "Photo library", and "Browse".</p><p><strong>Note!</strong> The image must be in JPG/JPEG, PNG, GIF, or SVG format, and the file size can be a maximum of 1 MB.</p><p>When the image upload is successful, it appears above the "<strong>1 Select image</strong>" icon. The learner can now click the lower button "<strong>2 Send selected image</strong>". If the file format is incorrect, an error message is displayed.</p><p>If the employee cannot upload their photo, they click the checkbox "<strong>I will not send my photo in advance</strong>".</p><p>When the employee has successfully sent the photo or decided not to send a photo, they click the arrow button on the right to continue.</p><p><strong>NOTE!</strong> If the employee is unable to send a photo of themselves, the access controller can take a photo of them on site.</p><h3>Element editing instructions</h3><p>Add a background image and edit all fields as needed.</p><p></p><h3>Element in published mode</h3><p><img src="statamic://asset::files::docs/2022/04/Upload-END-USER-FIN.gif"></p><p><img src="statamic://asset::files::docs/2022/04/Upload-Edit-FIN.gif"></p>
