<strong>The Pictures &amp; Captions </strong>element shows the steps to complete a particular task or process or can be used to check the details of a specific statement.

First, it explains how the element works.

On the right side of the element, there is a list of text boxes containing explanations of the topic.

A new image opens each time a new text box is clicked.

Each text box contains a short text explaining the corresponding image.

A "<em>Done</em>" sign appears on the text box when clicked.

Note! Each text box can be clicked more than once.

The sum of the text boxes is shown at the bottom right, and the topic title is at the top right.

<strong>Element e</strong><strong>diting instructions</strong>
<ul>
 <li>Plan the different steps.</li>
 <li>Open the element in <em>Edit </em>mode.</li>
 <li>Type the introduction in the text box. You can also leave the text box empty - in this case, the text box will not appear.</li>
 <li>Add each text box by clicking on the <em>Element settings</em> option in the orange wheel icon at the top of the element's edit view. The <em>Element settings</em> window opens.</li>
 <li>Add the required items, i.e. steps by clicking the "<em>Add item</em>" button.</li>
 <li>Each item or step will appear in <em>Edit</em> mode on the right side of the element.</li>
 <li>Click on the text box one item at a time. A new empty background image will open each time.</li>
 <li>Add a new background image to each item by clicking the <em>Change Image</em> icon in the top right corner of the background image.</li>
 <li>In each text box, add the text of the step.</li>
</ul>
