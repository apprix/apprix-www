<p>The <strong>Pictures &amp; Captions</strong> element shows the steps for completing a specific task or process, or it can be used to verify the details of a specific statement.</p><p>First, it is explained how the element works.</p><p>On the right side of the element there is a list with text boxes containing explanations of the topic in question.</p><p>A new image opens each time a new text box is clicked.</p><p>Each text box contains a short text explaining the corresponding image.</p><p>When a text box has been clicked, a "<em>Done</em>" mark appears on the screen.</p><p><strong>Note!</strong> The same text box can be clicked more than once.</p><p>At the lower right is the total count of text boxes, and at the upper right is the title of the topic in question.</p><h3>Element editing instructions</h3><ul><li><p>Plan the different steps.</p></li><li><p>Open the element in edit mode.</p></li><li><p>Write an introduction in the <em>Introduction</em> text box if you want to show an introduction at the start of this element. You can also leave the text box empty — in that case the text box will not be shown.</p></li><li><p>Each text box is added by clicking the <em>Element settings</em> option from the orange gear menu in the upper part of the element's edit view. The <em>Element settings</em> window opens.</p></li><li><p>Add the required items, i.e. steps, by clicking the "<em>Add item</em>" button.</p></li><li><p>Each item, i.e. step, appears on the right side of the element in <em>Edit</em> mode.</p></li><li><p>Click one text box on the right side at a time. A new empty background image opens each time.</p></li><li><p>Add a background image by clicking the <em>Change image</em> icon in the upper right corner of the background image.</p></li><li><p>Add the step text in each text box.</p></li></ul><p></p><h3><strong>Element in Preview mode</strong></h3><h3><img src="statamic://asset::files::docs/2023/07/Pictures-and-captions-preview-video-1.gif"></h3><h3><strong>Element in Edit mode</strong></h3><p><img src="statamic://asset::files::docs/2023/07/Pictures-and-captions-element-editing-video-1.gif"></p>
