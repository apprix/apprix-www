The <strong>Comments</strong> element allows the course facilitator to ask for open feedback and ideas from the course participants on the different topics covered during the training.

The course participant clicks on a chosen topic’s pen icon and writes their opinions, ideas, or feedback in the dropdown text box. After the participant has written their comment, the pen icon will be replaced by a Done icon to indicate that they have given their comment on that specific topic. Commenting on all topics is not mandatory. When the participant is ready, they can move to the next element by clicking the move forward arrow at the right side of the page. <strong> </strong>

<strong>Editing instructions</strong>
<ul>
 <li>Open the <strong>Comments</strong> element in <strong>Edit</strong> mode.</li>
 <li>If you wish to show an introduction at the beginning of this task, write the introduction and task instructions in the <strong>Introduction</strong> task box.
<ul>
 <li>You can also leave the Introduction task box empty. In this case, the box will not appear to the course participant.</li>
</ul>
</li>
 <li>Add the background image by clicking the <strong>Change Image</strong> icon at the right side of the window.</li>
 <li>Click the <strong>Element settings option</strong> from the gear icon at the top in <strong>the Edit Element contents</strong> view.</li>
 <li>Add all needed task boxes and drag them to the desired places.</li>
 <li>Now edit one task box at a time:
<ul>
 <li>Select the colour and the shape.</li>
 <li>Choose the pointer position and its shape.</li>
 <li>Edit the content (if you haven’t already from the gear icon).</li>
</ul>
</li>
</ul>
<ul>
 <li>Set additional <strong>Comments settings</strong> from the tool icon at the top left if needed.</li>
</ul>
<ul>
 <li>
<ul>
 <li><strong>Hide text box (only show pencil icon)</strong>:  hide subject texts and only show round pencil icons. <strong>Note!</strong> With this setting, you can place pen icons on top of any background image, illustration, or slide screenshot to ask for written feedback on specific parts.</li>
 <li><strong>Snap to grid: </strong>Move and resize subjects against the underlying grid. This feature makes it easier to arrange subjects together.</li>
</ul>
</li>
</ul>
<strong>Note!</strong>
<ul>
 <li>The "<strong>Rate comments</strong>" element is suggested to be added after the Comments element so that other course participants can read and rate each other’s ideas.</li>
 <li>Course participants can give their opinions on a specific argument several times.</li>
 <li>The maximum number of characters allowed for any text box is 500.</li>
 <li>The module can contain one or more than one Comments element.</li>
</ul>
<strong>How are comments restored, and in which cases are they not restored?</strong>

Comments left by the participant can be edited while they are in the element. However, they will not be returned when the participant returns to the element; in this case, the participant will leave new comments. Previous comments will still appear in the Rate Comments element. Only the most recent comment left by the student on the topic will be shown in the statistics.
