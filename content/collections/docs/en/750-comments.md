<p>The <strong>Comments</strong> element allows the course facilitator to ask learners for open feedback and ideas about various topics covered during the training.</p><p>The learner clicks the pencil icon for the selected topic and writes their opinion, idea, or feedback in the text field that opens. When the learner has written their comment in the text field, the pencil icon is replaced by a Done icon to indicate that they have expressed their opinion on that statement. Not all statements need to be commented on. When the learner is ready, they can move to the next element by clicking the forward arrow on the right side of the page.</p><h3>Element editing instructions</h3><ul><li><p>Open the Comments element in edit mode.</p></li><li><p>If you want to show an introduction at the start of this task, write the introduction and task instructions in the <strong>Introduction</strong> box.<ul><li><p>You can also leave the Introduction box empty. In that case the box will not be shown to the course participant.</p></li></ul></p></li><li><p>Add a background image by clicking the <strong>Change Image</strong> icon on the right side of the window.</p></li><li><p>Click the <strong>Element Settings</strong> option from the gear icon at the top of the <strong>Edit Element Contents</strong> view.</p></li><li><p>Add all the required text boxes and drag them to the desired positions.</p></li><li><p>Next, edit one text box at a time:<ul><li><p>Select the colour and shape.</p></li><li><p>Select the pointer position and shape.</p></li><li><p>Edit the content of the text box (unless you have already edited it from the element's gear icon settings).</p></li></ul></p></li></ul><ul><li><p>You can define other Comments element settings from the tool icon in the upper left corner:<ul><li><p><strong>Hide text box</strong> (only show pencil icon) - Hide the topic texts and show only the round pencil icons. <strong>Note</strong>! With this setting you can place pencil icons on top of any background image, illustration, or slideshow and request written feedback on specific parts.</p></li><li><p><strong>Snap to grid</strong> - move and resize topics against a background grid. This feature makes it easier to align topics with each other.</p></li></ul></p></li></ul><p><strong>Note!</strong></p><ul><li><p>The <strong>Rate Comments</strong> element is recommended to be added after the Comments element, so that other learners can read and rate each other's ideas.</p></li><li><p>Learners can submit their opinion on a specific statement multiple times.</p></li><li><p>The maximum allowed character count for the text field is 500 characters.</p></li><li><p>A module can contain one or more <strong>Comments</strong> elements.</p></li></ul><p><strong>How are comments returned, and in what situations are they not returned to the element?</strong></p><p>Comments left by the learner can be edited while in the element. However, they are not returned to the element when the learner revisits it — instead, the learner leaves new comments. Earlier comments are still visible in the Rate Comments element. The statistics only show the most recent comment left by the learner on a topic.</p><p></p><h3>How does a learner leave comments?</h3><p><img src="statamic://asset::files::docs/2025/03/Comment-1.gif"></p><h3>How is a text box edited?</h3><p><img src="statamic://asset::files::docs/2025/03/Com-task-box-1.gif"></p>
