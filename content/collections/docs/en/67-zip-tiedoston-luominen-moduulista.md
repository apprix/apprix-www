If the module needs to be exported from Builder to another environment or otherwise stored outside Builder, click the <strong>Link</strong> button. The "<strong>Published module link</strong>" pop-up window will open.

From here, you can:
- download the contents of the module as a zip file to your computer
- view the QR code
- copy or download the QR code of the module
- download the zip file (SCORM).
