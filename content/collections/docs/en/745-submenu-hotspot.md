<strong>The submenu hotspot </strong>element is similar to the <strong>Submenu button element</strong>, except the hotspots replace the text boxes at the right of the element and are placed above the background image as needed. <strong>In other words, each hotspot works as a submodule.</strong>

In the beginning, it is explained how the element works and how and in what order the learner should proceed through the chapters. A "<em>Done</em>" icon appears on the hotspot whenever a chapter is completed.

A pop-up window will inform the learner how to proceed when all chapters have been completed.

<strong>Note!</strong> The <em>Move Forward</em> button will appear based on the settings, and each chapter can be completed more than once.

<strong>Editing instructions</strong>
<ul>
 <li>Open the <em>Submenu hotspot.</em> The element must be in <em>Edit</em> mode <em>(" Edit Module &gt; Edit Element Contents"</em> window) to be edited.</li>
 <li>Open the <em>Image Library</em> by moving the mouse on the right side of the background image on top of the toolbar and clicking the lowest icon, <em>Change image</em>.</li>
 <li>In the new pop-up window, click <em>Browse / Upload Files…</em> Now you can see the Image Library. Choose the image to be displayed in the element.</li>
 <li>Edit the contents of the two pop-up windows opened in the element (Introduction and Summary). The Introduction pop-up window can be later found by clicking on the icon with the question mark. This appears as soon as the course participant opens the element. The bubble icon (Summary) will appear once the course participant has completed all the chapters on this page.</li>
 <li>Choose from the module's gear icon option "<em>Edit Submodule Menu Settings...</em>". The Submenu settings pop-up window is opened.</li>
 <li>Choose here the settings ("<em>Submodules have to be completed in the order defined below</em>" and "<em>All submodules have to be completed before being able to go forward</em>") and the number of buttons (hotspots) that you will need in the element by clicking the black "<em>Add button</em>."</li>
 <li>Rename each button (hotspot). The hotspots will appear at the upper left side of the element.</li>
 <li>Drag each hotspot and place it where it is needed.</li>
 <li>Adjust the hotspot size from its edges.</li>
 <li>Change the shape of the hotspot using the handle on the right side of the item.</li>
 <li>Move the mouse to the top of the left side of the hotspot. A brush will appear.</li>
 <li>Click the brush. A list of icons and colours you can use for the hotspot will appear on the left side of the window.</li>
 <li>Choose the icon and colour that you want for the hotspot.</li>
 <li>Repeat these steps for each hotspot.</li>
</ul>
<strong>And finally</strong>
<ul>
 <li>Link the submodules to the Submenu hotspot element</li>
 <li>Publish the submodules and the main module.</li>
</ul>
<strong> </strong>

<strong>How to link a submodule to a module?</strong>
<ul>
 <li>Create the module's content that will be later used as a submodule.</li>
 <li>Click in Edit mode on the "<em>Submodule settings</em>" button and then choose the "<em>Change to Submodule</em>" option.</li>
 <li>The "<em>Change to submodule</em>" pop-up window is opened. Click on the module where you would like to link your new submodule.</li>
 <li>The "<em>Submenu settings</em>" window is opened. Choose the item where you would like to link the submodule and click the correct "<em>Select button</em>." The new submodule is now linked to the main module.</li>
</ul>
