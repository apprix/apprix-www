<p>The <strong>Submenu Hotspot</strong> element is similar to the <strong>Submenu Buttons</strong> element, except that hotspots replace the text boxes on the right side of the element and are placed as needed on top of a background image. <strong>In other words, each hotspot acts as a submodule</strong>.</p><p>First, it is explained how the element works, as well as how and in what order the hotspots should be completed.</p><p>Whenever a hotspot has been completed, a "<em>Done</em>" icon appears on top of the hotspot icon.</p><p>When all hotspots have been completed according to the settings, a popup informs the participant how to continue the course.</p><p><strong>Note!</strong> The Move forward button appears based on the settings, and each hotspot can be completed more than once.</p><h3>Element editing instructions</h3><ul><li><p>Open the Submenu Hotspot element. The element must be in edit mode (<em>"Edit Module &gt; Edit Element Contents"</em> window) in order to be edited.</p></li><li><p>Open the image library by moving the mouse to the right side of the background image in the upper part of the toolbar and clicking the bottom icon, <em>Change image</em>.</p></li><li><p>In the new popup window, click <em>Browse / Upload Files...</em> You will now see the image library.</p></li><li><p>Select an image.</p></li><li><p>Edit the content of the two popup windows that open in the element (Introduction and Summary). The first window (Introduction) is found by clicking the icon with a question mark. It opens immediately when the element is clicked. The speech bubble icon (Summary) appears when all hotspots have been completed.</p></li><li><p>Select the "<em>Edit Submodule Menu Settings...</em>" option from the module's gear icon. The submenu settings popup opens.</p></li><li><p>Here, select the settings ("<em>Submodules have to be completed in the order defined below</em>" and "<em>All submodules have to be completed before being able to go forward</em>") and the number of hotspots needed in the element by clicking the black "<em>Add button</em>" button.</p></li><li><p>Name each button. The hotspots will appear in the upper left corner of the element.</p></li><li><p>Drag each hotspot and place it where it is needed.</p></li><li><p>Adjust the size of the hotspot from its edges.</p></li><li><p>Change the shape of the hotspot using the handle on the right side of the element.</p></li><li><p>Move the mouse to the upper left area of the hotspot. A brush will appear.</p></li><li><p>Click the brush. A list of icons and colours you can use in the hotspot appears on the left side of the window.</p></li><li><p>Select the icon and colour you want for the hotspot.</p></li><li><p>Repeat these steps for each hotspot.</p></li></ul><p><strong>Finally</strong></p><ul><li><p>Link the submodules to the Submenu Hotspot element.</p></li><li><p>Publish the submodules and the main module.</p></li></ul><p><strong>How do you link a submodule to a module?</strong></p><ul><li><p>Create the module content that will later be used as a submodule.</p></li><li><p>In edit mode, click the "<em>Submodules settings</em>" button and then select the "<em>Change to Submodule</em>" option.</p></li><li><p>The "<em>Change to submodule</em>" popup opens. Click the module to which you want to link the new submodule.</p></li><li><p>The "<em>Submenu settings</em>" window opens. Select the target to which you want to link the submodule and click the right "Select button". The new submodule is now linked to the main module.</p></li></ul><p><strong>Here is an example:</strong></p><p>Each cube with an icon contains a submodule. The learner must click the cubes and study their content. When each submodule is completed, a <em>Done</em> icon appears on the cube. When all submodules are completed, a popup with additional information appears and a <em>Move forward</em> button is visible on the right side of the window.</p><p></p><h3><strong>Element in published mode</strong></h3><p><img src="statamic://asset::files::docs/2023/07/Submenu-hotspot.jpg"></p><p><img src="statamic://asset::files::docs/2023/07/Submenu-hotspot-summary.jpg"></p><h3><strong>Element in edit mode</strong></h3><p><img src="statamic://asset::files::docs/2023/07/Submenu-hotspot.gif"></p><p><img src="statamic://asset::files::docs/2023/07/Submenu-hotspot-how-to-link-a-submodule-1.gif"></p>
