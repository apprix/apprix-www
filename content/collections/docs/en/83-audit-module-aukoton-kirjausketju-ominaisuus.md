An Audit Module (Audit Trail) is a chronological set of records that provide documentary evidence of the activities occurred at any time in a module.

You can activate the Audit Module option by clicking the module’s gear icon and by selecting Audit Module. If you need to disable the Audit trail, please select the option Set
<ul>
 <li><b>If you wish to activate Audit Module:</b> Click the module’s gear icon in<b> Edit Module </b>view and by selecting <b>Audit Module</b>. </li>
 <li><b>If you need to disable Audit Trail: </b>please select the option <b>Module Audit Off</b> from the module's gear icon again.<b> </b>When <b>Audit Module</b> is switched off,<b> </b>a pop up window informing this will appear.</li>
 <li>To view the recorded history of a module, please select <b>About this module</b> from the module’s gear icon in <b>Edit Module</b> view or from the Last modified... link on top of modules in <b>Menu</b> view.
<ul>
 <li>If the <b>Audit Module</b> option is switched on, the <b>About Module</b> pop-up window is displayed</li>
 <li>If the <b>Audit Module</b> is switched off, a pop-up window is displayed informing that the module auditing has not been enabled.</li>
</ul>
</li>
 <li>Please note that it is possible also to export the collected data in PDF, CSV, and Excel format.</li>
</ul>
In the picture on the right side, you can see that the updates done to module’s texts and images are collected in the <b>Module Change History</b>.

<h4>Tracked information</h4>
Below is a detailed list of what kind of information is tracked when <b>Audit Trail </b>is active:
<ul>
 <li>Element’s edited content (e.g. text, pictures, videos)</li>
 <li>Added, deleted and arranged elements</li>
 <li>Renamed element’s and/or module’s title</li>
 <li>Disabled/enabled personal password</li>
 <li>Published/unpublished modules</li>
 <li>Duplicated modules</li>
</ul>
<strong>Please, remember to reload Builder's page so that the Administrate -&gt; Client -&gt; Features -&gt; Audit / Full on Publishing (to display archived files also incremental to be on) setting is enabled.</strong>
