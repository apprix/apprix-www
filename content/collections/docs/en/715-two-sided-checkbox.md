<p><strong>Two-Sided Checkbox</strong> is a task element and is commonly used in tests together with the <strong>Pass / Fail</strong> element.</p><p>The learner must select the correct statement from two options. A text box with additional information is displayed after each selection.</p><h3>Element editing instructions</h3><ul><li><p>Write the task introduction in the introduction box if you want to show an introduction at the start of the task. You can also leave the box empty. In that case, the introduction box will not be visible to the learner.</p></li><li><p>Add a background image by clicking the <strong>Change image</strong> icon in the upper right corner of the element.</p></li><li><p>Click <strong>Element settings</strong> from the gear menu at the top of the element's edit view.</p></li><li><p>Add the desired number of fields from the <strong>Element settings</strong> window.</p></li><li><p>Edit the statements.</p></li></ul><p><strong>"Answer Mode" functionality</strong></p><p>If you want to activate the <strong>Answer Mode</strong> functionality, click the gear icon on the left edge of the window.</p><p><strong>Option 1: Answer Mode is turned off</strong></p><ul><li><p>Click the checkbox of each item. The answer text popup appears.</p></li><li><p>Edit its content.</p></li><li><p>Write additional information about the topic of the statement here. Note that the content is the same regardless of the answer given (e.g. Left/Right).</p></li></ul><p><strong>Option 2: Answer Mode is turned on</strong></p><ul><li><p>Enable Answer Mode by clicking the blue button in the settings bar.</p></li><li><p>Now click the checkbox of the item, and the answer text opens in a popup. CORRECT (Right) and INCORRECT (Wrong) options appear at the top of the popup. Provide justifications for each option.</p></li></ul><p><strong>NOTE!</strong> If the answer text in the popup is left empty, the answer text window will not appear when the learner has selected the answer checkbox.</p><h3>Element in preview mode</h3><p><img src="statamic://asset::files::docs/2022/04/Two-sided-checkbox-END-USER-ENG.gif"></p><p><img src="statamic://asset::files::docs/2022/04/Two-sided-checkbox-Editor-ENG.gif"></p>
