The <strong>T</strong><b>wo-Sided Checkbox</b> is an exercise element that is commonly used in tests with the <b>Pass</b> / <b>Fail</b> element.

The employee must choose the correct statement from two options. A text box with additional information is displayed after each selection. 
<h4>Editing Instructions</h4>
<ul>
 <li aria-level="1">Write the introduction and task instructions in the <b>Introduction</b> box, if you wish to show an introduction at the beginning of this task.
<ul>
 <li aria-level="2">You can also leave the box empty. In this case, the box will not appear to the employee.</li>
</ul>
</li>
 <li aria-level="1">Add the background image by clicking the <b>Change Image</b> icon from the top right corner of the element.</li>
 <li aria-level="1">Click the <b>Element settings</b> option from the gear icon at the top of the element.
<ul>
 <li aria-level="2">In the pop-up window, add the needed items and choose the correct option.</li>
</ul>
</li>
 <li aria-level="1">Edit the statements.</li>
</ul>
<b>“Answer Mode” Options</b>

If you want to activate the <b>Answer Mode</b> option, click the gear icon on the left side of the window.

<b>Option 1. The Answer Mode is switched off</b>
<ul>
 <li aria-level="1">Click each item tick box. An answer text pop-up window will appear. </li>
 <li aria-level="1">Edit its content.</li>
 <li aria-level="1">Write here additional information about the topic in the statement. Note, that the content will be the same regardless of the given answer (e.g. Left/Right).</li>
</ul>
<b>Option 2: The Answer Mode is switched on</b>
<ul>
 <li aria-level="1">Switch on the <b>Answer Mode</b> by clicking the blue button on the Setting Bar. </li>
 <li aria-level="1">Click on the item tick box, and the <b>Answer text</b> pop-up window will be opened. On top of the pop-up window, the <b>RIGHT</b> and <b>WRONG</b> options will appear. Type the answers for the different options.</li>
</ul>
<b>NOTE! </b>If the <b>Answer text </b>pop-up window is left empty, no pop-up window will be displayed after the employee has ticked the answer box.
