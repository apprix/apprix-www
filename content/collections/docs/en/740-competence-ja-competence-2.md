<p>In this element, an employee can add their valid competence certificates (e.g. safety card, electrical safety card, hot work card) from a competence list requested by the employer.</p><p>The employee adds a card/competence by clicking the "+" icon on the left. A dropdown menu opens from which the predefined cards can be selected. After selecting a card, the employee fills in the card number and its validity period (month and year).</p><p><strong>Note!</strong> The card number is a free text field (letters can be used if needed).</p><p>A selected card can be removed by clicking the trash can button. When all cards and their related information have been entered, the employee can click the forward button.</p><p><strong>Note!</strong> The employee can only select one card at a time, meaning the content of the dropdown menu shrinks as options are already selected.</p><p>The last item in the dropdown menu is "<strong>Something else, what?</strong>", which is a text field where the employee can enter a card that is not listed in the dropdown menu.</p><p><strong>Note!</strong> For each competence there is a "mandatory" or "optional" choice. If a competence is "mandatory", Builder displays it directly in the element and the employee cannot remove it by pressing the trash can button.</p><h3>Element editing instructions</h3><ul><li><p>Fill in the fields and provide the necessary instructions for the employee.</p></li></ul><p>For example:</p><p><strong><em>Fill in your competence details</em></strong></p><p><em>To work on this construction site, you must have at least a valid safety card.</em></p><p><em>Fill in the card numbers and validity periods below. When you are ready, press the forward arrow to continue.</em></p><ul><li><p>Select the <strong>Element Settings</strong> option from the gear icon at the top</p></li><li><p>Add and edit the required items (the text will appear on the left edge of the window).</p></li></ul><p>For example:</p><ul><li><p><ul><li><p><em>Safety card</em></p></li><li><p><em>Electrical safety card</em></p></li><li><p><em>Hot work card</em></p></li></ul></p></li></ul><ul><li><p>Update the content of the "<strong>Something else, what?</strong>" item if needed.</p></li><li><p>If needed, select the "<strong>Mandatory</strong>" option from the item's tool icon.</p></li></ul><p><strong>NOTE!</strong> If a card/competence is "<strong>mandatory</strong>", the item is shown directly in the element and the employee cannot remove it by pressing the trash can button.</p><ul><li><p>Click the Move forward arrow. A message box will appear. This box is shown to the employee if one or more certificates have expired when they click the Move forward arrow.</p></li><li><p>Write instructions for the employee in case one or more certificates have expired.</p></li></ul><p><strong>Note!</strong></p><p>Data entered in the Competence element is not saved to statistics if the module does not use authentication methods such as a Login element, email registration, SSO, or tax number verification.</p><h3>Competence 2</h3><p>Competence 2 is an improvement of the Competence element.</p><p>In the upper left corner of the Competence 2 element there is a wrench icon. Click it. The Competence options window opens. Here you can select the following options for all rows:</p><ul><li><p>Card number column</p></li><li><p>All cards at once</p></li><li><p>"Other, what?" as the last row.</p></li></ul><p></p><h3><em>The employee adds their valid competence certificates from the competence list requested by the employer.</em></h3><p><img src="statamic://asset::files::docs/2022/04/Competence1.gif"></p><h3><em>List of competences.
