In this element, the employee can add their valid certificates of qualification (e.g. <i>Occupational Safety Card, Electrical Safety Certificate, Fire Work License</i>)  from a list of qualifications the employer requests. 

The employee clicks the plus icon on the left to add a card/qualification. A drop-down menu opens as a modal from which predefined qualifications can be selected. After the card selection, the employee fills in the card number and validity period (month and year). 

<b>Note!</b> The card number is a free text field (letters can be entered, if necessary)

Clicking the trash button deletes the selected card. Once all cards and related information have been entered, the employee can click the forward button.

<b>Note!</b> The employee can select only one qualification at a time, so the content of the drop-down menu is reduced depending on whether the options have already been selected.

The last item in the drop-down menu is <b>“Something else, what?”</b> a text field where the employee can enter a specific qualification not listed in the menu.

<b>Note!</b> Each qualification has a “mandatory” or “optional” option. If the qualification is “mandatory,” then the Builder will display it directly in the element, and the employee will not be able to delete it by pressing the trash button since the trash icon is not available in case the item is mandatory.
<h4 class="fusion-responsive-typography-calculated" data-fontsize="21" data-lineheight="31.5px">Editing Instructions</h4>
<ul>
 <li aria-level="1">Fill in the fields and give the needed instructions to the employee. </li>
</ul>
<p>E.g.: </p>
<p><b><i>Fill out your qualifications</i></b></p>
<p><i>To be able to work in this area, you must have at least a valid occupational safety card.</i><i>
</i><i>Fill out the card number and their validity period below. When ready, press the Move forward arrow to proceed.</i></p>

<ul>
 <li aria-level="1">Choose the <b>Element Settings</b> option from the gear icon at the top </li>
 <li aria-level="1">Add and edit the required items (the text will appear on the left side of the window). </li>
</ul>
<p>E.g.: </p>

<ul>
 <li>
<ul>
 <li aria-level="2">Occupational Safety Card  </li>
 <li aria-level="2">Electrical safety certificate</li>
 <li aria-level="2">Firework license </li>
</ul>
</li>
 <li aria-level="1">Update the content of the item ”Jokin muu, mikä?” (”Something else, what?” ), if needed.</li>
 <li aria-level="1">Select from the item’s tool icon the option ”<b>Mandatory</b>” if needed.</li>
</ul>
<p><b>NOTE!</b> Suppose the card/qualification is “mandatory.” In that case, the item is displayed directly in the element, and the employee cannot delete it by pressing the trash button.</p>

<ul>
 <li aria-level="1">Click on the <b>Move forward</b> arrow. A message box is displayed. This box shows the employee if one or more certificates have expired after clicking the "<b>FORWARD</b>" button.</li>
 <li aria-level="1">Write the instructions for the employee in case one or more certificates have expired.</li>
</ul>
<strong>Note</strong>!

The data entered in the Competence element is not saved in the statistics if the module does not use a user authentication method, for example, a Login element, Email Registration, SSO, or Tax Number Check.
<h4>Competence 2</h4>
Competence 2 is an enhancement of the Competence element.

A wrench icon is at the top left of the Competence 2 element.  Click it. The Competence options window will be opened. Here, you can select the following options for all rows:
<ul>
 <li>Card number column</li>
 <li>All cards at once</li>
 <li>"Other, what?" as the last row.</li>
</ul>
