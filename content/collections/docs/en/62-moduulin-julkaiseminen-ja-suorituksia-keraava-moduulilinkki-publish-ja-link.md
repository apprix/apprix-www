<p>When you are satisfied with the content of your module, publish it by clicking the <strong>Publish</strong> button in the top-right corner of the <strong>Edit Module</strong> view. The text on the Publish button changes to <strong>Published</strong>. A <strong>Stats button</strong> appears to the right of the <strong>Link</strong> button, where you can view the module's completion statistics.</p><p>If the training contains a main module and its submodules, use the main module's link as the completion link.</p><p><strong>Note!</strong> Do not share the link opened from the Preview button with course learners, as it allows skipping over module elements/pages and does not save completion statistics. This feature is intended for preview purposes only. You can recognise the Preview link by its ending: <strong>&amp;skip&amp;nostats</strong>.</p><p><strong>Note!</strong> Cancel the module's publication by clicking the <strong>Published</strong> button to toggle it off. If you have the <strong>Drafting</strong> feature enabled, you will also need to choose whether to apply or discard the changes made in the draft version (Draft version).</p><h3>Sharing the published (Published) link</h3><p>The published module link is found behind the Link button. The Link button becomes active once the module has been published.</p><p>The link menu contains:</p><ul><li><p>The published module link.</p></li><li><p>The link's QR code.</p></li><li><p>The published module zip package (SCORM).</p></li></ul><p>From this link menu you can copy the module link and QR code, download the QR code and the zip file (SCORM).</p><h3>How do you publish a module and copy its link?</h3><p><img src="statamic://asset::files::docs/2024/07/Publish-link.gif"></p><h3>How do you share the published link from the "All Modules" view?</h3><p><img src="statamic://asset::files::docs/2024/07/Published-link-all-modules.gif"></p><h3>What is the difference between the published link and the preview link?</h3><p><img src="statamic://asset::files::docs/2025/06/Publish-preview-1.gif"></p>
