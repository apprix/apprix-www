When you are satisfied with the content of your module, publish it by clicking the <strong>Publish</strong> button in the upper right corner of the <strong>Edit Module</strong> view. The text of the <strong>Publish</strong> button changes to <strong>Published</strong>. A <strong>Stats</strong> button appears to the right of the <strong>Link</strong> button, from where you can view the module's performance statistics.

If the training includes the main module and its submodules, provide the main module link as the e-learning link.

<strong>Note!</strong> Do not share the <strong>Preview</strong> link with the eLearning participants, as it allows them to jump over the module elements/pages, and does not save their performance data in the statistics. The Preview link is intended to be used for preview purposes only. You can recognize the Preview link by its suffix: <strong>&amp;skip&amp;nostats</strong>.

<strong>Note!</strong> Unpublish the module by turning the <strong>Published</strong> button off.

If you have the <strong>Drafting</strong> feature enabled, you have also to choose whether to save or reject the draft version before unpublishing the module.

<strong> </strong>
<h4><strong>Sharing the Published Link</strong></h4>
The published module link can be found behind the <strong>Link</strong> button. The link button is activated when the module is published.

The <strong>Link</strong> menu includes:
<ul>
 <li>Published module link.</li>
 <li>QR code of the link.</li>
 <li>Published module zip package (SCORM).</li>
</ul>
From this link menu, you can copy the link and QR code of the module, and download the QR code and the module’s zip file (SCORM).
