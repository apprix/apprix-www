---
<strong>Switch</strong> is a task element. In this element, the trainee has to answer questions or statements by choosing one of the three available options, e.g. "<b>Never</b>"-"<b>Sometimes</b>"-"<b>Always</b>". After the trainee has answered all the questions/statements, s/he clicks the move forward arrow button on the right side of the page. The correct answer will appear for each question including a message containing further information about the question topic. The trainee can check the details of the questions again by pressing the “i” icon.
<h4>Editing Instructions</h4>
<ul>
 <li>In the introduction box, write the exercise instructions and their purpose. If you leave the box empty, it will not be visible to the trainee. </li>
 <li>Add a background image by clicking the <b>Change image</b> icon on the right side of the element. </li>
 <li>Click the <b>Element settings</b> option from the gear icon at the top of <strong>Edit Element Contents</strong> view. The <b>Element Settings</b> window will be opened. 
<ul>
 <li>Add all the needed items. Edit the questions/statements for the exercise. </li>
 <li>Add the option texts on top of the switches, e.g. "Never"-"Sometimes"-"Always"</li>
 <li>Set the correct options.</li>
</ul>
</li>
 <li>It is also possible to add an image below each switch option text.</li>
 <li>In the <b>Info</b> icon next to each text box, write any additional information related to the topic. This text content is shown after the trainee has provided their answers. </li>
 <li>Click the orange <b>tool</b> icon on the top left of the element window. The <b>Settings Bar</b>, which allows you to change the color of the text and answer boxes, will be displayed. Change the colors according to your preference.</li>
</ul>
