<p>In the <strong>Switch</strong> task element, the learner must answer questions or statements by clicking to select one of three available options on a slider scale, for example: &quot;Never&quot; - &quot;Sometimes&quot; - &quot;Always&quot;. When the learner has answered all of the task's questions/statements, they click the forward button on the right side of the page. An answer appears one at a time for each question, containing additional information about the topic. The learner can review the additional information for questions again by pressing the i-info icon.</p><h3>Element editing instructions</h3><ul><li><p>Write the task instructions for learners in the introduction box — that is, how the task works and what its purpose is. If you leave the box empty, it will not be shown to the course participant.</p></li><li><p>Add a background image by clicking the <strong>Change image</strong> icon in the upper right part of the element.</p></li><li><p>Click the <strong>Element Settings</strong> option from the gear icon in the upper part of the element's edit view. The <strong>Element Settings</strong> window opens. Add as many sliders as you need. Edit the task questions/statements. Add the answer options above the sliders, e.g. &quot;Never&quot; - &quot;Sometimes&quot; - &quot;Always&quot;, and finally set the correct answer options in the <strong>Element Settings</strong> window.</p></li><li><p>It is also possible to add an image below the text associated with each slider.</p></li><li><p>Write all relevant additional information in the Info icon next to each text field. This content will appear after the learner has given their answer and clicked the forward button.</p></li><li><p>Click the orange tool icon in the upper left corner of the window. A settings bar for changing the colour of the text and answer boxes will appear. Select the desired colour for the boxes.</p></li></ul><p></p><h3>Element in published mode</h3><p><img src="statamic://asset::files::docs/2021/08/Switch_FI-1.gif"></p><h3>Element settings in edit view</h3><h3><img src="statamic://asset::files::docs/2021/08/Switch_Edit_FI-1.gif"></h3><h3>Switch element in use</h3><p><img src="statamic://asset::files::docs/2021/06/Switch1.png"></p><p><img src="statamic://asset::files::docs/2021/06/Switch2.png"></p>
