title: '6.1 Previewing a Module (Preview)'

<p>To see the module you have built from the learner's perspective, click the <strong>Preview</strong> button in the bottom-left corner of the module in the <strong>All modules</strong> view. The preview opens in a new tab. Go through the entire module and edit elements in <strong>Edit</strong> mode as needed. In the preview, you can navigate forwards or backwards using the buttons in the red '<strong>Preview mode</strong>' box visible in the bottom corner. You can access the module editing view by clicking the <strong>Edit</strong> button from the <strong>All Modules</strong> view on the front page.</p><p>Note that there is a difference between the <strong>Preview</strong> buttons in the <strong>All Modules</strong> view and the <strong>Edit Module</strong> view:</p><ul><li><p>In the <strong>All Modules</strong> view: the <strong>Preview</strong> button displays the module directly in a new tab.</p></li><li><p>In the <strong>Edit Module</strong> view: the <strong>Preview</strong> button opens a dropdown menu with two options: <strong>Preview Share</strong> and <strong>Preview Open</strong>.</p></li></ul><p><strong>Note!</strong> If you have added submodules to your module, you must publish (Publish) them before previewing. A green dot will appear next to published submodules in the submodule structure of the editing view.</p><p><strong>Note!</strong> You can recognise a <strong>Preview</strong> link by its ending: <strong>&amp;skip&amp;nostats</strong>.</p><h3>Sharing the preview link</h3><p>In the dropdown menu of the <strong>Preview</strong> button in the Edit module view, you will find two different options:</p><ul><li><p><strong>Open</strong>: Select this option when you want to open the module preview in a new tab.</p></li><li><p><strong>Share</strong>: Select this option when you want to share the module preview link with external users for testing purposes.</p></li></ul><p><strong>Note!</strong></p><ul><li><p>If you want to share the main module and all its submodules, open the Share popup from the main module.</p></li><li><p>If you want to share only one submodule, open the Share popup from that submodule.</p></li></ul><p><strong>How do I share the module link further for testing purposes from the Share module preview popup?</strong></p><ul><li><p>From the <strong>Edit Module</strong> window, select <strong>Preview</strong> &agrave; <strong>Share</strong></p></li><li><p>Activate the <strong>Share</strong> feature by clicking the <strong>Enable preview sharing</strong>… button.</p></li><li><p>Enter a password that is at least 6 characters long and contains lowercase letters, uppercase letters, and numbers. You can also use the &quot;<strong>Generate password</strong>&quot; button.</p></li><li><p>Select an end date for the preview from the calendar. It can be up to one month from today, and you can change the end date later if needed.</p></li><li><p>Click the <strong>Copy link and message</strong> button and forward its content to an external user for testing purposes.</p></li><li><p>Click the Save button.</p></li></ul><p>After this, the preview does not require any further authentication.</p><p>The <strong>IP Guard</strong> feature restricts access to the preview link for external testers. For external testers to successfully access the preview, the IP Guard feature must be disabled during testing.</p><p><strong>Remember to re-enable the disabled IP Guard feature after testing.</strong></p><h3>How to preview a module?</h3><p><img src="statamic://asset::files::docs/2025/06/Preview-module-feature.gif"></p><h3>How to preview an element using the 'Preview Element in New Tab' option?</h3><p><img src="statamic://asset::files::docs/2025/06/Preview-element-in-new-tab.gif"></p><h3>How do you save and share the preview link?</h3><p><img src="statamic://asset::files::docs/2024/07/Preview-share-link.gif"></p>
