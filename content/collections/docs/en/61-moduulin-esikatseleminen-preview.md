---
To preview the module from the course participant’s perspective, click the <strong>Preview</strong> button in the bottom left corner of the module within the '<strong>All modules</strong>' view. The preview will open in a new tab. Review the entire module, and if necessary, leave a comment on the page by clicking the green bubble on the right side of the window or edit the elements in Edit mode. While previewing, you can navigate forward or backwards using the buttons in the red '<strong>Preview mode</strong>' box located in the bottom left corner. You can return to the module’s editing view by clicking the <strong>Edit</strong> button in the <strong>All Modules</strong> view on the homepage.

<strong>Note!</strong> You can recognise the Preview link by its suffix: <strong><em>&amp;skip&amp;nostats</em></strong>.

Note that there is a difference between the <strong>Preview</strong> buttons in the <strong>All Modules</strong> and <strong>Edit Module</strong> views:
<ul>
 <li>In the <strong>All Modules view</strong>, the Preview button displays the module directly in a new tab.</li>
 <li>In the <strong>Edit Module view</strong>, the Preview button contains a drop-down menu with two options: <strong>Preview Share</strong> and <strong>Preview Open</strong>.</li>
</ul>
<strong>Note!</strong>  A submodule structure is an e-learning course that contains the main module + one or more submodules. All submodules must be published before you can preview them in the <strong>Submenu buttons</strong> element in the main module. Publish the submodules by opening each in the <strong>Edit Module view</strong> and clicking the <strong>Publish </strong>button<strong>. </strong>A light green circle will appear next to published submodules in the submodule menu list.

<strong> </strong>
<h4><strong>Preview Link (“Open” and ”Share”) options</strong></h4>
There are two different options in the drop-down menu of the <strong>Preview</strong> button in the <strong>Edit module</strong> view:
<ul>
 <li><strong>Open</strong>: Select this option to open the module preview in a new tab.</li>
 <li><strong>Share</strong>: Select this option when you want to deliver the module's preview link to external users for testing purposes.</li>
</ul>
<strong> </strong><strong>Note!</strong>
<ul>
 <li>If you wish to share the main module and all its submodules, open the sharing pop-up window in the main module.</li>
 <li>If you wish to share only one submodule, open the sharing pop-up window in that submodule.</li>
</ul>
<h4></h4>
<h4><strong>How do I share the module link from the Share module preview pop-up window for testing purposes?</strong></h4>
<ul>
 <li>Select from the <strong>Edit Module</strong> view module's  <strong>Preview</strong> -&gt; <strong>Share</strong></li>
 <li>Activate the <strong>Share</strong> feature by clicking the <strong>Enable preview sharing</strong>… button.</li>
 <li>Fill in a <strong>password</strong> that is at least six characters long, including lower-case letters, upper-case letters, and numbers. You can also use the "<strong>Generate password</strong>" button.</li>
 <li>Select an end date from the calendar. It can be up to one month from today, and you can change it later if needed.</li>
 <li>Select the <strong>Copy link and message</strong> button, and forward its content to an external user for testing.</li>
 <li>Select the <strong>Save</strong></li>
</ul>
After these steps, the preview does not require any further identification from the external user.

The <strong>IP Guard feature</strong> restricts external testers' use of the preview link. Therefore, the feature must be disabled during testing for external users to preview the module.

<strong><u>Be sure to re-enable the disabled IP Guard feature after testing!</u></strong>
