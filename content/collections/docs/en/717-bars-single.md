<p>The <strong>Slider Bars Single</strong> element can be used as a survey or for collecting signals. In the element, the learner can evaluate many, possibly related, topics. The goal is to collect the learner's opinions and assessments of the topics, and the element has no correct answers.</p><p>The learner is shown a popup with a question. The learner moves the bar to respond to the given question. All bars appear on screen one at a time.</p><p>The selected values are saved to the statistics.</p><p><strong>Tip!</strong> If the entire module or the specific element covers a topic that may cause resistance, it is good to start with a positive approach. Ask the learner how much the features of the change can benefit them in their tasks. By presenting the change in this way, the learner will also adopt a positive attitude.</p><h3>Element editing instructions</h3><p>Add a background image by clicking the <strong>Change Image</strong> icon in the upper right corner of the element.</p><p>Click <strong>Element Settings</strong> from the gear icon at the top. Add the required bars to the popup. New bars appear in the element. Write a title at the bottom of each bar.</p><p>Now write the question / scenario and ask the user to move the bars to the positions corresponding to their assessment.</p><p>Click the "info" icon at the bottom left of each bar. A popup opens. Write the bar text here. You can add a lot of information to the popup. This content is shown before the user moves the slider.</p><p>Add an element title if needed.</p><h3>Bars Survey element</h3><p>The Bars Survey element is similar to the Bars Slider Single element. It allows the learner to evaluate different topics.</p><p>The facilitator can define in the element settings whether the average of other participants' answers is shown to the learner at the end of the exercise.</p><h3>Element in published mode</h3><p><img src="statamic://asset::files::docs/2021/08/Bars_Slider_Single_FI_.gif"></p><h3>Element settings in edit view</h3><p><img src="statamic://asset::files::docs/2021/08/Bars_Slider_Single_Edit_FI_.gif"></p><h3><img src="statamic://asset::files::docs/2022/04/7.16-1-1-600x324.png">Bars Survey options</h3><p></p><p><img src="statamic://asset::files::docs/2024/12/Bars-Survey-options.png"></p><h3>Bars Survey - Average of other participants' answers</h3><p><img src="statamic://asset::files::docs/2024/12/Bars-Survey-Statistics.png"></p>
