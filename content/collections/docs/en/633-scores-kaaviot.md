<h3>Slider bars task</h3><p>This chart presents the correct, average, and maximum values of the responses.</p><p>Chart header, on the left side:</p><ul><li><p>Element name in blue (the text visible above the element in the Edit Module view)</p></li><li><p>Element name in bold (slider_bars_task)</p></li><li><p>Element page number (#n).</p></li></ul><p>Top-right corner of the chart</p><ul><li><p>"x completed" – how many completions are included in the element with the filter active.</p></li><li><p>An info button opens a modal when you hover the mouse cursor over it. The modal describes the element's content.</p></li></ul><p>Chart</p><ul><li><p>The bars represent the average of all learners who have completed the course.</p></li><li><p>The green line and number represent the correct answer set for the bar.</p></li><li><p>The text below each bar refers to the title set for the individual bar in the element's Edit view.</p></li><li><p>Hovering the mouse cursor over a bar (or tapping on a mobile device) reveals a modal showing the exact value for the average and maximum.</p></li></ul><h3>Slider bars single</h3><p>This chart presents the average and maximum values of the responses.</p><p>Chart header, on the left side:</p><ul><li><p>Element name in orange (the text visible above the element in the Edit Module view)</p></li><li><p>Element name in bold (slider_bars_single)</p></li><li><p>Element page number (#n).</p></li></ul><p>Top-right corner of the chart</p><ul><li><p>"x completed" – indicates how many completions are included in the element with the active filter.</p></li><li><p>An info button opens a modal when you hover the mouse cursor over it. The modal describes the card.</p></li></ul><p>Chart</p><ul><li><p>The bar average represents the average of all course learners' responses.</p></li><li><p>The text below each bar refers to the title set for the individual bar in the element's Edit view.</p></li><li><p>Hovering the mouse cursor over the orange or light grey bar (or tapping on a mobile device) reveals a modal showing the exact average and maximum values.</p></li></ul><h3>Fourfold</h3><p>This fourfold element displays learners' responses in a heatmap style. The more dots overlap, the stronger the colour shown.</p><p>Chart header, on the left side:</p><ul><li><p>Element name in orange (the text visible above the element in the Edit Module view)</p></li><li><p>Element name in bold (fourfold)</p></li><li><p>Element page number (#n).</p></li></ul><p>Top-right corner of the chart</p><ul><li><p>"x completed" – indicates how many completions are included in the element with the active filter.</p></li><li><p>An info button opens a modal when you hover the mouse cursor over it. The modal describes the card.</p></li></ul><p>Chart</p><ul><li><p>The legends above the chart indicate what colour each statement's responses appear in the chart, and clicking them shows or hides the corresponding statement's responses.</p></li><li><p>When you hover the mouse cursor over a dot, it shows how many dots are at the same position (answers: n) and what the coordinates of the dot are.</p></li></ul><h3>Slider icons</h3><p>The Slider icons card shows how responses are distributed on a scale of 0–10. When you hover the mouse cursor over a bar, it shows how many people answered e.g. the number 6. The leftmost bar represents response number 0 and the rightmost represents response number 10. The minimum possible value is 0 and the maximum is 10.</p><p>Chart header, on the left side:</p><ul><li><p>Element name in orange (the text visible above the element in the Edit Module view)</p></li><li><p>Element name in bold (slider_icons)</p></li><li><p>Element page number (#n).</p></li></ul><p>Top-right corner of the chart</p><ul><li><p>"x completed" – indicates how many completions are included in the element with the active filter.</p></li><li><p>An info button opens a modal when you hover the mouse cursor over it. The modal describes the card.</p></li></ul><p>Chart</p><ul><li><p>The text below the horizontal bar shows each question from the element's Edit view.</p></li><li><p>Below the titles, tabs are visible representing the different questions of the element. You can browse the responses to the different questions of the element by switching tabs.</p></li><li><p>The orange horizontal bar represents the combined average score of all x respondents.</p></li><li><p>When you hover the mouse cursor over a bar, a modal appears showing the value corresponding to the bar and the number of respondents for it.</p></li><li><p>The text below the horizontal bar is from each question's Edit view.</p></li></ul><h3>Slider write text</h3><p>The Slider write text (slider_ability) card shows how responses are distributed on a scale of 1–5. When you hover the mouse cursor over a bar, it shows how many people answered e.g. the number 2. The leftmost bar represents response number 1 and the rightmost represents 5. The minimum possible value is 1 and the maximum is 5.</p><p>Chart header, on the left side:</p><ul><li><p>Element name in orange (the text visible above the element in the Edit Module view)</p></li><li><p>Element name in bold (slider_ability)</p></li><li><p>Element page number (#n).</p></li></ul><p>Top-right corner of the chart</p><ul><li><p>"x completed" – how many completions are included in the element with the filter active.</p></li><li><p>An Info button opens a modal when you hover the mouse cursor over it. The modal describes the element's content.</p></li></ul><p>Chart</p><ul><li><p>Below the titles, tabs are visible representing the different questions of the element. You can browse the responses to the different questions of the element by switching tabs.</p></li><li><p>The orange horizontal bar represents the combined average score of all x respondents.</p></li><li><p>When you hover the mouse cursor over a bar, a modal appears showing the value corresponding to the bar and the number of respondents for it.</p></li><li><p>The text below the horizontal bar is from each question's Edit view.</p></li></ul><h3>Tips!</h3><ul><li><p>An element is also removed from the Scores view if you delete it from the module.</p></li><li><p>If a user deletes and adds questions to an element in an already-published module, the statistics may become distorted. We recommend creating a new element and deleting the old one.</p></li><li><p>If a user modifies an element in an already-published module, the statistics may distort the correct answers.</p></li><li><p>If you are searching for data with filters that do not match any completions, a clear message will appear on screen.</p></li></ul><h3>Chart - Slider bars task</h3><p><img src="statamic://asset::files::docs/2023/03/8-Slider-bars-task-1.png"></p><h3>Chart - Slider bars single</h3><p><img src="statamic://asset::files::docs/2023/03/9-Slider-bars-single-1.png"></p><h3>Chart - Fourfold</h3><p><img src="statamic://asset::files::docs/2023/03/10-Fourfold-1.png"></p><h3>Chart - Slider icons</h3><p><img src="statamic://asset::files::docs/2023/03/11-Slider-icons-1.png"></p><h3>Chart - Slider write text (slider_ability)</h3><p><img src="statamic://asset::files::docs/2023/03/12-Slider-write-text.png"></p>
