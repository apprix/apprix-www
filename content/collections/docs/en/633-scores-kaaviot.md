<h4><strong>Slider bars task</strong></h4>
This chart shows the correct, average, and maximum values ​​of the given answers.

Chart header left margin:
<ul>
 <li>The element name is in blue (the text displayed above the element in the Edit Module view)</li>
 <li>The element name in bold (slider_bars_task)</li>
 <li>Element page number (#n).</li>
</ul>
Chart upper right corner
<ul>
 <li>“n completed” – how many course completions are included in the element (the number can be different depending on the used filter).</li>
 <li>The info button opens a modal when the mouse cursor is placed over it. The modal shows further information about the card.</li>
</ul>
Chart
<ul>
 <li>The bars reflect the average of all those who completed the course.</li>
 <li>The green line and number placed on the bar represent the correct answer.</li>
 <li>The text below each bar refers to the title set for the individual bar in the Edit view of the element.</li>
 <li>When placing the mouse cursor over the bar (or pressing with a mobile device), the modal that appears tells the average and maximum values.</li>
</ul>

<h4><strong>Fourfold</strong></h4>
This chart displays the responses of the course participants in a heatmap style. The more spots are on top of each other, the stronger appears the spot color.

Chart header left margin:
<ul>
 <li>The element name is in orange (text visible above the element in the Edit Module view)</li>
 <li>The element name in bold (fourfold)</li>
 <li>Element page number (#n).</li>
</ul>
Chart upper right corner
<ul>
 <li>“n completed” – how many course completions are included in the element (the number can be different depending on the used filter)</li>
 <li>The info button opens a modal when the mouse cursor is placed over it. The modal shows further information about the card.</li>
</ul>
Chart
<ul>
 <li>The legends above the diagram indicate the color of the answers to each statement in the diagram, and by pressing them, you can show or hide the answers to each statement.</li>
 <li>When you move the mouse cursor over a spot, it shows how many spots are in the same position (answers: n) and what are their coordinates.</li>
</ul>
<h4><strong>Slider icons</strong></h4>
The Slider icons chart shows how the answers are distributed on a scale of 0-10. When you move the mouse cursor over the bar, it shows how many course participants have answered in each case, e.g. two-course participants have given the answer 6. The leftmost bar reflects the answer number 0 and the rightmost one 10. The lowest possible value is 0 and the highest is 10.

Chart header left margin:
<ul>
 <li>The element name is in orange (text visible above the element in the Edit Module view)</li>
 <li>The element name in bold (slider_icons)</li>
 <li>Element page number (#n).</li>
</ul>
Chart upper right corner
<ul>
 <li>“n completed” – how many course completions are included in the element (the number can be different depending on the used filter)</li>
 <li>The info button opens a modal when the mouse cursor is placed over it. The modal shows further information about the card.</li>
</ul>
Chart
<ul>
 <li>The text below the horizontal bar is the name of each question given in the Edit module view of the element.</li>
 <li>The tabs shown below the headings are the different questions for the element. You can browse the answers to the different questions in the element by changing the tab.</li>
 <li>The orange horizontal bar describes the average score of all “n” course participants.</li>
 <li>When you hover over it, a modal appears showing the value corresponding to the bar and the number, of the course participants who answered it.</li>
 <li>The text below the horizontal bar is the name of each question from the Edit module view of the element.</li>
</ul>
<h4><strong>Slider write text</strong></h4>
Slider writes text (slider_ability) chart shows how the answers are distributed on a scale of 1-5. When you move the mouse cursor over the bar, it shows how many course participants have answered e.g. the number 2. The leftmost bar reflects the answer number 1 and the rightmost 5. The smallest possible value is 1 and the largest is 5.

Chart header left margin:
<ul>
 <li>The element name is in orange (text visible above the element in the Edit Module view)</li>
 <li>The element name in bold (slider_ability)</li>
 <li>Element page number (#n).</li>
</ul>
Chart upper right corner
<ul>
 <li>“n completed” – how many course completions are included in the element (the number can be different depending on the used filter)</li>
 <li>The info button opens a modal when the mouse cursor is placed over it. The modal shows further information about the card.</li>
</ul>
Chart
<ul>

<h4><strong>Tips!</strong></h4>
<ul>
 <li>The element is removed from the <strong>Scores</strong> view if you remove it from the module.</li>
 <li>If the facilitator changes the content of a task element in an already published module, the statistics of correct answers may be distorted. We suggest you create a new element and delete the old one.</li>
 <li>If you use a filter that gives no results, a clear message will be shown on the screen.</li>
</ul>
