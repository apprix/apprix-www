For example, this element can be used in the final test of a course when it is necessary to test how well the user has learned the module's concepts, such as speed limits, safety distances, heights, lengths, and widths.

The element contains a series of questions (<strong>a maximum of 8 questions</strong>) that the user must answer by entering a value (number or text) in a <strong>white text box</strong> to the right of the question box.

Once the user has answered all the questions, a ”<strong>Next</strong>” button will appear on the right-hand side of the element. The user must click the button. To the left of the top question box, an answer text box with a comment and the correct value will appear. If the user has answered correctly, the row will be <strong>green</strong>. If the course participant has answered incorrectly, the row will appear in <strong>red</strong>.

A Next button is on the right side of the answer text box. When the course participant clicks the button, the answer text box will close, and the lower answer text box will appear. The course participant must repeat these steps until the last answer text box appears. When the course participant has reviewed the content of all the answer text boxes, the Next button will appear on the right-hand side of the element. Clicking this button will take the student to the next element.

<strong>Editing instructions</strong>

Start by adding a background image to the element. Then click <strong>Element Settings</strong> from the gear icon. Add the required number of items.

In the left text box, explain how the exercise works. On the right side of the element, starting from the left, enter the question, the correct answer (number or text) in the white box, and finally, the unit, for example, <strong>km/h, m, cm</strong>, etc.

Then click on the text field of the top question box. A text box with an arrow on the right side will appear. Type a comment and the correct value for the user.

Repeat these steps for each row.

Click the <strong>tool icon</strong> at the top left of the element, and then <strong>choose whether you want to allow only numbers or numbers and letters in the white box</strong>.

<strong>Note that in the white box:</strong>
<ul>
 <li>Commas and periods are considered to be the same character.</li>
 <li>If you only allow numbers, zeros after the decimal separator (comma/period) will be considered the same number, for example, "1.100" and "1.1", and "1,100" and "1.1".</li>
 <li>The answer must be correct if you allow characters and numbers, including the number of zeros after a period/comma.</li>
</ul>
