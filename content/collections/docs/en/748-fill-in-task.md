<p>This element can be used, for example, in the final test of a course, when it is necessary to test how well the user has learned the things taught in the module, such as speed limits, safety distances, heights, lengths, and widths.</p><p>The element contains a series of questions (up to 8 questions), which the user must answer by entering a value (a number or text and a number) into the <strong>white text field</strong> on the right side of the question box.</p><p>When the learner has answered all the questions, a "<strong>Forward</strong>" button appears on the right side of the element. The learner must now click the button. An answer text box appears to the left of the top question box, containing a comment and the correct value. If the learner answered correctly, the row is green. If the learner answered incorrectly, the row is red.</p><p>On the right side of the answer text box there is a <strong>Next</strong> button. When the learner clicks the button, the answer text box closes and the next answer text box appears. The learner must repeat these steps until the last answer text box. When the learner has gone through the content of all the answer text boxes, the "Forward" button appears on the right side of the element. By clicking this button, the learner moves to the next element.</p><p></p><p><strong>Element editing instructions</strong></p><p>Start by adding a background image to the element. Then click <strong>Element Settings</strong> from the gear icon. Add the required number of items.</p><p>Explain how the exercise works in the left text field. On the right side of the element, starting from the left, write the question, enter the correct answer (number or text and number) in the white box, and finally the unit, for example <strong>km/h, m, cm</strong>, etc.</p><p>Then click the text field of the top question box. A text field with an arrow will appear. Write a comment and the correct value for the user.</p><p>Repeat these steps for each row.</p><p>Then click the tool icon in the upper left corner of the element and select whether you want to allow only numbers or both numbers and letters in the white box.</p><p></p><p><strong>Please note that in the white box:</strong></p><ul><li><p>Commas and periods are treated as the same character.</p></li><li><p>If you allow only numbers, trailing zeros after the decimal separator (comma/period) are treated as the same number, for example "1,100" and "1,1" as well as "1,100" and "1,1".</p></li><li><p>If you allow characters other than numbers, the answer must be exactly correct, including the number of zeros after the period/comma.</p></li></ul><p></p><h3>Element in published mode</h3><p><img src="statamic://asset::files::docs/2024/11/Fill-in-in-Published-mode-1.jpg"></p><h3>The user checks that the answers are correct.</h3><p><img src="statamic://asset::files::docs/2024/11/Fill-in-in-Published-mode.jpg"></p><h3>Element in edit mode</h3><p><img src="statamic://asset::files::docs/2024/11/Fill-in-in-Edit-mode.jpg"></p><h3>Do you want to allow only numbers or both numbers and letters in the white boxes?</h3><p><img src="statamic://asset::files::docs/2024/11/Fill-in-task-settings.png"></p>
