---
In the <strong>Image Library,</strong> you can add and remove the images you need in the module elements.
<h4><strong>Images format and size instructions</strong></h4>
<ul>
 <li>JPG format is suitable for images. Full background images should be 1600x900 or 1920x1080 pixels (JPG quality 8).</li>
 <li>You can also upload larger JPG files. If the file size is over 550kb (max 20 MB), they will be automatically optimised for web use.</li>
 <li>PNG format also allows vector images, graphs, slides, and screenshots of user interfaces (or transparent images). If an uploaded PNG file is over 550kb, it will be converted to JPG.</li>
</ul>
