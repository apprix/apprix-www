<p>In the image library (<strong>Image Library</strong>) you can add and remove images that you need for the module's elements.</p><h3>Guidelines for image format and size</h3><ul><li><p>JPG format is suitable for images. Full background images should be <strong>1600x900</strong> or <strong>1920x1080</strong> pixels in size (<strong>JPG quality 8</strong>).</p></li><li><p>You can also upload larger JPG files. If the file size is <strong>over 550 KB</strong> (up to a maximum of <strong>20 MB</strong>), it is automatically optimised for web use.</p></li><li><p>PNG format can also be used for vector images, diagrams, slides, and screenshots of user interfaces (or transparent images). If an uploaded <strong>PNG</strong> file is <strong>over 550 kilobytes</strong>, it will be converted to <strong>JPG</strong> format.</p></li></ul><h3>Where can the Image Library be found?</h3><p><img src="statamic://asset::files::docs/2024/04/5.3-Open-image-library.png"></p>
