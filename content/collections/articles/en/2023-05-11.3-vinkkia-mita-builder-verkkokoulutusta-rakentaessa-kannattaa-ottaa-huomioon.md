        text: 'Tip 1: Break Information into Bite-Sized Pieces'
  -
    type: paragraph
    content:
      -
        type: text
        text: 'Information should not be served all at once to the point of overload. Instead, informational sections should be broken into easily digestible bite-sized pieces, making it easier to internalise the content. Builder has a built-in feature that sets the font size and the amount of text per page so that you do not accidentally overwhelm the learner with too much information at once.'

    type: heading
    attrs:

        text: 'Tip 2: Use Interim Exercises'
  -
    type: paragraph
    content:
      -
        type: text
        text: 'Interim exercises are a great way to break up a course into sections and pace the learning. The task does not even need to be large – a simple "True or False?" statement is enough to prompt the learner to think about the topic independently, bringing their focus back to the course at hand if their mind starts to wander during reading sections. Builder has several different exercise types, and it is worthwhile to use them diversely in courses to improve both learning and concentration.'

  -
    type: paragraph
    content:
      -
        type: text
        text: 'Take into account different types of learners – one learns by reading, another by doing exercises, and a third internalises things best from an illustrative video. Most people benefit from all of these, so it is well worth making generous use of Builder''s various element types. Would you try an element type today that you have never tested before?'
