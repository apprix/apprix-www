        type: text
        text: 'My first contact with Apprix Builder was through courses others had created with it, and I saw a few before I started using the tool myself. My first impression was that the courses are really clear for the end user. I immediately knew where to click to advance in the course and what to click to answer a task.'
  -
    type: paragraph
    content:
      -
        type: text
        text: 'Soon I got to practice using the tool myself – actually on the very first day of my internship. When I first saw the view where courses are created, it seemed a bit confusing and there appeared to be a lot of elements. Once I got to try it out myself and understood the basic principles of how the elements work, Builder quickly became clearer and I learned to choose the right element for each situation.'
  -
    type: paragraph
    content:
      -
        type: text
        text: 'On the second day of my internship, I started creating a practice course on a topic that interests me – sourdough baking. I boldly tried to use as many elements as possible, because I noticed that you learn best by doing and experimenting. After working on the course for a while, it started to feel enjoyable and new ideas about different possibilities began to emerge. I thought it would take much more time to learn, but I mastered the basics within the first week.'
  -
    type: paragraph
    content:
      -
        type: text
        text: 'During the first month of my internship, in addition to my own practice course, I have already been able to create language versions for clients. Creating language versions is quite simple work, mainly copying and pasting, but through it I have also gained more experience and confidence in using Builder. I have hardly encountered any difficulties or problems with Builder, apart from a few minor things that were quickly resolved by asking others.'
  -
    type: paragraph
    content:
      -
        type: text
        text: 'I would also like to encourage other Builder beginners to boldly try different elements and use them diversely in their courses. The short user training I received provides a good foundation and opportunity to get started. You cannot learn everything during it, but experimenting is worthwhile and teaches you a lot.'
  -
    type: paragraph
    content:
      -
        type: text
        text: 'I still have two months left of my internship, and I believe I will learn a lot more during that time. As you may have already gathered, I find using Builder technically very easy. However, I believe there is still plenty to learn in the coming months as I get to create more content myself and also think about the pedagogical side. I hope that going forward I will get to brainstorm content ideas more and use my creativity.'
